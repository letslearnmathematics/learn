Add & Subtract Fractions


Let's now learn how to add and subtract fractions.

Addition of Fractions


Let's first consider fractions who have the same denominator.





Fractions with the same denominator

When fractions have the same denominator, you only have to add the numerators and maintain the denominator.

Example 1


Solve \(\frac{2}{9} + \frac{4}{9}\)

Solution


\(\hspace{0.5cm}\frac{2}{9} + \frac{4}{9}\)

\(\Rightarrow \frac{2 \ + \ 4}{9}\)

\(\Rightarrow \frac{6}{9}\)

\(\Rightarrow \frac{2 \times 3}{3 \times 3}\)

\(\Rightarrow \frac{2}{3}\)





Example 2


Simplify the expression: \(\frac{2}{3} + \frac{5}{3}\)

Solution


\(\frac{2}{3} + \frac{5}{3}\)

\(\Rightarrow \frac{2 \ + \ 5}{3}\)

\(\Rightarrow \frac{7}{3}\)

\(\Rightarrow 2\frac{1}{3}\)





Example 3


Simplify: \(\frac{1}{7} + \frac{2}{7} + \frac{4}{7}\)

Solution


\(\frac{1}{7} + \frac{2}{7} + \frac{4}{7}\)

\(\Rightarrow \frac{1 \ + \ 2 \ + \ 4}{7}\)

\(\Rightarrow \frac{7}{7}\)

\(\Rightarrow 1\)





Remember:

When adding fractions with the same denominator, maintain the denominator, and then add the numerators.
