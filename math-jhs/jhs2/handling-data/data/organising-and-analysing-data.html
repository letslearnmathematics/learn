Introduction

Data organization is the process of arranging collected data in a meaningful way to make it easier to understand, analyze, and interpret. Properly organized data helps us identify patterns, trends, and relationships.

In this lesson, we will learn how to organize both grouped and ungrouped data using various representations including frequency tables, bar graphs, pie charts, waffle diagrams, and stem-and-leaf plots.

Grouped Data:

Data organized into classes or intervals (e.g., 10-19, 20-29)

Ungrouped Data:

Raw data showing individual values (e.g., 12, 15, 18, 21)

Waffle Chart

Indicator 3: The Waffle Chart

A waffle chart is a square grid that shows proportions of a whole, where each small square represents a portion of the total.

Create a waffle chart to represent the frequency of scores 15 and 16 from our data set (total of 8 occurrences out of 20).

Waffle chart representation (assuming 100 squares = 20 scores):

Calculation:

  • Each square represents 0.2 scores (20 scores ÷ 100 squares)
  • Score 15 (5 scores) = 25 squares
  • Score 16 (3 scores) = 15 squares
  • Remaining scores = 60 squares

Stem and Leaf Plot

Indicator 4: Stem and Leaf Plot

A stem-and-leaf plot displays quantitative data by splitting each value into a "stem" (first digit(s)) and a "leaf" (last digit).

12, 15, 18, 21, 22, 24, 25, 25, 27, 28, 31, 32, 33, 35, 36, 38, 41, 42, 45, 47
