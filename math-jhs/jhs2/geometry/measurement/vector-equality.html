Introduction to Vector Equality

Vector equality refers to the condition when two vectors are considered identical. Two vectors are equal if and only if they have the same magnitude and the same direction.

In mathematical terms, vectors \(\vec{a}\) and \(\vec{b}\) are equal (\(\vec{a} = \vec{b}\)) if:

\[ \begin{cases} |\vec{a}| = |\vec{b}| & \text{(same magnitude)} \\ \text{Direction of } \vec{a} = \text{Direction of } \vec{b} & \text{(same direction)} \end{cases} \]

Graphical Representation:

Equal vectors can be represented by arrows that are parallel, have the same length, and point in the same direction, regardless of their position in space.

Component Form:

For vectors in component form \(\vec{a} = \langle a_1, a_2 \rangle\) and \(\vec{b} = \langle b_1, b_2 \rangle\), they are equal if and only if:

\[ a_1 = b_1 \text{ and } a_2 = b_2 \]

Properties of Equal Vectors

Reflexive Property

Any vector is equal to itself:

\[ \vec{a} = \vec{a} \]

Symmetric Property

If one vector is equal to another, then the second is equal to the first:

\[ \text{If } \vec{a} = \vec{b} \text{, then } \vec{b} = \vec{a} \]

Transitive Property

If a vector is equal to a second vector, and the second is equal to a third, then the first is equal to the third:

\[ \text{If } \vec{a} = \vec{b} \text{ and } \vec{b} = \vec{c} \text{, then } \vec{a} = \vec{c} \]

Addition Property

Equal vectors maintain equality when the same vector is added to both:

\[ \text{If } \vec{a} = \vec{b} \text{, then } \vec{a} + \vec{c} = \vec{b} + \vec{c} \]

Scalar Multiplication Property

Equal vectors maintain equality when both are multiplied by the same scalar:

\[ \text{If } \vec{a} = \vec{b} \text{, then } k\vec{a} = k\vec{b} \text{ for any scalar } k \]

Examples of Vector Equality

Example 1: Component Verification

Determine if the vectors \(\vec{u} = \langle 3, -2 \rangle\) and \(\vec{v} = \langle 3, -2 \rangle\) are equal.

Solution:

Since both vectors have identical components:

\[ \vec{u} = \langle 3, -2 \rangle \] \[ \vec{v} = \langle 3, -2 \rangle \]

Therefore, \(\vec{u} = \vec{v}\).

Example 2: Different Positions

Consider two vectors with the same magnitude and direction but different initial points. Are they equal?

Solution:

Yes, vectors are equal regardless of their position in space as long as they have:

  • The same magnitude (length)
  • The same direction

Position doesn't affect vector equality.

Example 3: Magnitude and Direction

Vector \(\vec{a}\) has magnitude 5 units and direction 30° north of east. Vector \(\vec{b}\) has magnitude 5 units and direction 30° north of east. Are they equal?

Solution:

Yes, since both vectors have:

  • Same magnitude (5 units)
  • Same direction (30° north of east)

Therefore, \(\vec{a} = \vec{b}\).

Practice Exercise

Question 1

Which of the following vectors are equal to \(\vec{v} = \langle -2, 5 \rangle\)?

a) \(\vec{a} = \langle -2, 5 \rangle\)

b) \(\vec{b} = \langle 2, -5 \rangle\)

c) \(\vec{c} = \langle -2, 5, 0 \rangle\) (in 3D space)

Solution:

a) Equal - same components

b) Not equal - opposite components

c) Not equal - different dimensions (2D vs 3D)

Question 2

If \(\vec{p} = \vec{q}\) and \(\vec{q} = \vec{r}\), what can you conclude about \(\vec{p}\) and \(\vec{r}\)?

Solution:

By the transitive property of vector equality:

\[ \text{If } \vec{p} = \vec{q} \text{ and } \vec{q} = \vec{r} \text{, then } \vec{p} = \vec{r} \]

Question 3

Vector \(\vec{u}\) has magnitude 10 units pointing west. Vector \(\vec{v}\) has magnitude 10 units pointing east. Are they equal?

Solution:

No, they are not equal. While they have the same magnitude (10 units), they have opposite directions (west vs east).

For vectors to be equal, both magnitude and direction must be identical.

Question 4

If \(\vec{m} = \langle x, y \rangle\) and \(\vec{n} = \langle 3, -4 \rangle\) are equal, what are the values of x and y?

Solution:

For the vectors to be equal, corresponding components must be equal:

\[ x = 3 \] \[ y = -4 \]

Question 5

True or False: Two vectors with the same magnitude but different directions can be equal.

Solution:

False. For vectors to be equal, they must have both the same magnitude and the same direction.

Applications of Vector Equality

Navigation

Vector equality is crucial in navigation systems where direction and magnitude of movement must be precisely matched to reach a destination.

Physics and Engineering

In physics, equal force vectors produce identical effects regardless of their point of application, as long as they have the same magnitude and direction.

Computer Graphics

In computer graphics and game development, vector equality is used to determine when objects are moving in the same direction with the same speed.
