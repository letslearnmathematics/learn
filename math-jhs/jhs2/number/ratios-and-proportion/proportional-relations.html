Introduction

A proportional relationship between two quantities means they change at the same rate. When one quantity changes, the other changes in a predictable way that maintains their ratio.

In this lesson, we'll learn how to:

  1. Use tables to check if two quantities have a proportional relationship
  2. Use graphs to identify proportional relationships

Example: If 2 pens cost Gh₵5, then 4 pens cost Gh₵10, 6 pens cost Gh₵15, etc.

This shows a proportional relationship because the ratio of cost to number of pens is always the same (5:2).

Key Characteristics

A proportional relationship has these features:

  • The ratio between the two quantities is always the same (constant ratio)
  • When graphed, it forms a straight line that passes through the origin (0,0)
  • It can be represented by the equation y = kx, where k is the constant of proportionality

Using Tables to Check Proportional Relationships

Method 1: Equivalent Ratios

To determine if two quantities are proportional using a table:

  1. Calculate the ratio of y/x for each pair of values
  2. If all the ratios are equal, the relationship is proportional
  3. If the ratios are not equal, the relationship is not proportional

Example: Does this table show a proportional relationship?

Number of Books (x) Price in Gh₵ (y) Ratio y/x
2 10 10/2 = 5
3 15 15/3 = 5
5 25 25/5 = 5
8 40 40/8 = 5

Since all ratios equal 5, this is a proportional relationship.

Non-Proportional Example

Example: Does this table show a proportional relationship?

Time (hours) Distance (km) Ratio y/x
1 50 50/1 = 50
2 90 90/2 = 45
3 130 130/3 ≈ 43.33
4 150 150/4 = 37.5

Since the ratios are not equal, this is NOT a proportional relationship.

Calculate the ratios:

12/3 = 4, 20/5 = 4, 28/7 = 4, 36/9 = 4

All ratios equal 4, so this is a proportional relationship.

The ratios are not equal, so this is NOT a proportional relationship.

Using Graphs to Check Proportional Relationships

Characteristics of Proportional Graphs

A graph shows a proportional relationship if:

  1. It is a straight line
  2. The line passes through the origin (0,0)

  • It's a straight line
  • It passes through (0,0)

Real-World Applications

Where We See Proportional Relationships

Proportional relationships are everywhere in daily life:

  • Currency exchange: The amount you receive is proportional to the amount you exchange
  • Recipes: Ingredients are proportional to the number of servings
  • Speed: Distance traveled is proportional to time when speed is constant
  • Sales: Total cost is proportional to number of items when price per item is fixed

Shopping Example

Scenario: A shop sells notebooks at Gh₵3 each. The total cost is proportional to the number of notebooks bought.

We can represent this with:

  1. A table showing number of notebooks and total cost
  2. A graph plotting these values
  3. An equation: Total Cost = 3 × Number of Notebooks (y = 3x)

Practice Exercise

Question 1

A car travels at a constant speed of 60 km/h. Create a table showing the distance traveled after 1, 2, 3, and 4 hours. Is this a proportional relationship?

Time (hours) Distance (km) Ratio y/x
1 60 60
2 120 60
3 180 60
4 240 60

Yes, this is a proportional relationship because all ratios equal 60.

Question 2

A phone plan charges Gh₵50 per month plus Gh₵1 per minute of calls. Create a table for 10, 20, 30, and 40 minutes. Is this a proportional relationship?

Minutes (x) Total Cost (y) Ratio y/x
10 60 6
20 70 3.5
30 80 ≈2.67
40 90 2.25

No, this is NOT a proportional relationship because the ratios are not equal.
