Introduction

An exponential equation is an equation in which the variable appears in the exponent. To solve these equations, we often use the laws of indices to express both sides with the same base or in a comparable form.

The general form of an exponential equation is: \[ a^x = b \] where \( a \) is the base, \( x \) is the exponent (which contains the variable we need to solve for), and \( b \) is a constant or another exponential expression.

Key Strategy:

To solve exponential equations:

  1. Express both sides with the same base when possible
  2. Set the exponents equal to each other
  3. Solve the resulting equation

Equations with Same Base

Fundamental Principle

If two exponential expressions with the same base are equal, then their exponents must be equal: \[ \text{If } a^m = a^n \text{, then } m = n \text{ (where } a > 0 \text{ and } a \neq 1\text{)} \]

Example 1:

Solve \( 3^{2x} = 3^{x+4} \)

Solution:

Since the bases are equal (both are 3), we set the exponents equal:

\[ 2x = x + 4 \] \[ 2x - x = 4 \] \[ x = 4 \]

Example 2:

Solve \( 5^{x-2} = 5^{3x+1} \)

Solution:

Set the exponents equal since bases are the same:

\[ x - 2 = 3x + 1 \] \[ -2 - 1 = 3x - x \] \[ -3 = 2x \] \[ x = -\frac{3}{2} \]

Practice Exercise

Question 1

Solve \( 2^{x+3} = 2^{5x-1} \)

Since bases are equal (2), set exponents equal:

\[ x + 3 = 5x - 1 \] \[ 3 + 1 = 5x - x \] \[ 4 = 4x \] \[ x = 1 \]

Question 2

Solve \( 7^{2x-1} = 7^{x+4} \)

Set exponents equal since bases are same (7):

\[ 2x - 1 = x + 4 \] \[ 2x - x = 4 + 1 \] \[ x = 5 \]

Question 3

Solve \( 10^{3x+2} = 10^{x-5} \)

Set exponents equal since bases are same (10):

\[ 3x + 2 = x - 5 \] \[ 3x - x = -5 - 2 \] \[ 2x = -7 \] \[ x = -\frac{7}{2} \]

Equations with Different Bases

Conversion Strategy

When bases are different but can be expressed as powers of the same base, we rewrite them accordingly:

Example 1:

Solve \( 4^x = 8^{x+1} \)

Solution:

Express both sides with base 2:

\[ (2^2)^x = (2^3)^{x+1} \] \[ 2^{2x} = 2^{3x+3} \]

Now set exponents equal:

\[ 2x = 3x + 3 \] \[ -x = 3 \] \[ x = -3 \]

Example 2:

Solve \( 9^{2x} = 27^{x-1} \)

Solution:

Express both sides with base 3:

\[ (3^2)^{2x} = (3^3)^{x-1} \] \[ 3^{4x} = 3^{3x-3} \]

Set exponents equal:

\[ 4x = 3x - 3 \] \[ x = -3 \]

Important Note

When the bases cannot be expressed as powers of the same base, you may need to use logarithms. However, at the JHS level, we typically encounter problems where bases can be converted to match.

Practice Exercise

Question 1

Solve \( 8^x = 16^{x+1} \)

Express with base 2:

\[ (2^3)^x = (2^4)^{x+1} \] \[ 2^{3x} = 2^{4x+4} \] \[ 3x = 4x + 4 \] \[ -x = 4 \] \[ x = -4 \]

Question 2

Solve \( 25^{x-1} = 125^{2x} \)

Express with base 5:

\[ (5^2)^{x-1} = (5^3)^{2x} \] \[ 5^{2x-2} = 5^{6x} \] \[ 2x - 2 = 6x \] \[ -4x = 2 \] \[ x = -\frac{1}{2} \]

Question 3

Solve \( 16^{2x} = 64^{x-3} \)

Express with base 4:

\[ (4^2)^{2x} = (4^3)^{x-3} \] \[ 4^{4x} = 4^{3x-9} \] \[ 4x = 3x - 9 \] \[ x = -9 \]

Equations in Quadratic Form

Substitution Method

Some exponential equations can be transformed into quadratic equations using substitution:

Example 1:

Solve \( 2^{2x} - 5 \cdot 2^x + 4 = 0 \)

Solution:

Let \( y = 2^x \), then \( 2^{2x} = y^2 \):

\[ y^2 - 5y + 4 = 0 \] \[ (y - 1)(y - 4) = 0 \]

So \( y = 1 \) or \( y = 4 \)

Substitute back:

\[ 2^x = 1 \Rightarrow x = 0 \] \[ 2^x = 4 \Rightarrow x = 2 \]

Solutions: \( x = 0 \) or \( x = 2 \)

Example 2:

Solve \( 3^{2x} - 4 \cdot 3^x + 3 = 0 \)

Solution:

Let \( y = 3^x \), then \( 3^{2x} = y^2 \):

\[ y^2 - 4y + 3 = 0 \] \[ (y - 1)(y - 3) = 0 \]

So \( y = 1 \) or \( y = 3 \)

Substitute back:

\[ 3^x = 1 \Rightarrow x = 0 \] \[ 3^x = 3 \Rightarrow x = 1 \]

Solutions: \( x = 0 \) or \( x = 1 \)

Word Problems

Problem-Solving Approach

Many real-world situations involve exponential growth or decay that can be modeled with exponential equations. The key steps are:

  1. Identify the exponential relationship in the problem
  2. Set up the appropriate equation
  3. Solve the equation using the methods we've learned
  4. Interpret the solution in the context of the problem

Example 1:

A bacteria culture doubles every 3 hours. If there are 500 bacteria initially, how long will it take for the population to reach 8,000?

Solution:

The growth model is: \( P = P_0 \cdot 2^{t/3} \)

\[ 8000 = 500 \cdot 2^{t/3} \] \[ 16 = 2^{t/3} \] \[ 2^4 = 2^{t/3} \] \[ 4 = \frac{t}{3} \] \[ t = 12 \text{ hours} \]

Example 2:

A car depreciates at 15% per year. If its current value is $20,000, how many years will it take for the value to reach $10,000?

Solution:

The depreciation model is: \( V = V_0 \cdot (0.85)^t \)

\[ 10000 = 20000 \cdot (0.85)^t \] \[ 0.5 = (0.85)^t \]

This requires logarithms to solve exactly, but at JHS level we might estimate:

After 4 years: \( (0.85)^4 \approx 0.522 \)

After 5 years: \( (0.85)^5 \approx 0.444 \)

So it takes between 4 and 5 years to reach half value.

Practice Exercise

Question 1

A population of rabbits triples every 2 months. If there are 50 rabbits initially, how long will it take for the population to reach 1,350 rabbits?

Growth model: \( P = 50 \cdot 3^{t/2} \)

\[ 1350 = 50 \cdot 3^{t/2} \] \[ 27 = 3^{t/2} \] \[ 3^3 = 3^{t/2} \] \[ 3 = \frac{t}{2} \] \[ t = 6 \text{ months} \]

Question 2

A radioactive substance decays by 20% each hour. How many hours will it take for 1000g to reduce to 512g?

Decay model: \( A = 1000 \cdot (0.8)^t \)

\[ 512 = 1000 \cdot (0.8)^t \] \[ 0.512 = (0.8)^t \]

Recognize that \( 0.8^3 = 0.512 \)

\[ t = 3 \text{ hours} \]

Question 3

An investment grows at 5% per year. How many years will it take for the investment to double?

Growth model: \( A = P \cdot (1.05)^t \)

\[ 2P = P \cdot (1.05)^t \] \[ 2 = (1.05)^t \]

This requires logarithms for exact solution, but we can estimate:

After 14 years: \( (1.05)^{14} \approx 1.98 \)

After 15 years: \( (1.05)^{15} \approx 2.08 \)

So it takes about 14-15 years to double.
