Introduction

Skip counting is a method of counting numbers by skipping forward or backward by a fixed number other than 1. This technique helps in developing number sense and understanding patterns in numbers.

In this lesson, we will learn how to skip count forwards and backwards in large increments such as 10,000s, 100,000s, and 500,000s. This skill is particularly useful for working with large numbers in real-world applications.

Counting Forward

Counting Forward in 500,000s

To count forward in 500,000s, we start with a given number and repeatedly add 500,000 to get the next numbers in the sequence.

Example 1:

Count forward in 500,000s up to the fifth number starting from 200,000.

Solution:

  1. Start with 200,000
  2. Add 500,000: 200,000 + 500,000 = 700,000
  3. Add 500,000: 700,000 + 500,000 = 1,200,000
  4. Add 500,000: 1,200,000 + 500,000 = 1,700,000
  5. Add 500,000: 1,700,000 + 500,000 = 2,200,000

The sequence is: 200,000, 700,000, 1,200,000, 1,700,000, 2,200,000

Example 2:

Count forward in 100,000s up to the fifth number starting from 50,000.

Solution:

  1. Start with 50,000
  2. Add 100,000: 50,000 + 100,000 = 150,000
  3. Add 100,000: 150,000 + 100,000 = 250,000
  4. Add 100,000: 250,000 + 100,000 = 350,000
  5. Add 100,000: 350,000 + 100,000 = 450,000

The sequence is: 50,000, 150,000, 250,000, 350,000, 450,000

Counting Backward

Counting Backward in 100,500s

To count backward in 100,500s, we start with a given number and repeatedly subtract 100,500 to get the next numbers in the sequence.

Example 1:

Count backward in 100,500s up to the fifth number starting from 1,800,000.

Solution:

  1. Start with 1,800,000
  2. Subtract 100,500: 1,800,000 - 100,500 = 1,699,500
  3. Subtract 100,500: 1,699,500 - 100,500 = 1,599,000
  4. Subtract 100,500: 1,599,000 - 100,500 = 1,498,500
  5. Subtract 100,500: 1,498,500 - 100,500 = 1,398,000

The sequence is: 1,800,000, 1,699,500, 1,599,000, 1,498,500, 1,398,000

Example 2:

Count backward in 50,000s up to the fifth number starting from 500,000.

Solution:

  1. Start with 500,000
  2. Subtract 50,000: 500,000 - 50,000 = 450,000
  3. Subtract 50,000: 450,000 - 50,000 = 400,000
  4. Subtract 50,000: 400,000 - 50,000 = 350,000
  5. Subtract 50,000: 350,000 - 50,000 = 300,000

The sequence is: 500,000, 450,000, 400,000, 350,000, 300,000

Practice Exercise

Question 1

Count forward in 500,000s up to the fifth number starting from 300,000.

Solution:

  1. 300,000
  2. 300,000 + 500,000 = 800,000
  3. 800,000 + 500,000 = 1,300,000
  4. 1,300,000 + 500,000 = 1,800,000
  5. 1,800,000 + 500,000 = 2,300,000

The sequence is: 300,000, 800,000, 1,300,000, 1,800,000, 2,300,000

Count forward in 100,000s up to the fifth number starting from 75,000.

Solution:

  1. 75,000
  2. 75,000 + 100,000 = 175,000
  3. 175,000 + 100,000 = 275,000
  4. 275,000 + 100,000 = 375,000
  5. 375,000 + 100,000 = 475,000

Count backward in 100,500s up to the fifth number starting from 2,000,000.

Solution:

  1. 2,000,000
  2. 2,000,000 - 100,500 = 1,899,500
  3. 1,899,500 - 100,500 = 1,799,000
  4. 1,799,000 - 100,500 = 1,698,500
  5. 1,698,500 - 100,500 = 1,598,000

The sequence is: 2,000,000, 1,899,500, 1,799,000, 1,698,500, 1,598,000

Question 4

Count backward in 50,000s up to the fifth number starting from 750,000.

Solution:

  1. 750,000
  2. 750,000 - 50,000 = 700,000
  3. 700,000 - 50,000 = 650,000
  4. 650,000 - 50,000 = 600,000
  5. 600,000 - 50,000 = 550,000

The sequence is: 750,000, 700,000, 650,000, 600,000, 550,000

Question 5

Count forward in 250,000s up to the fifth number starting from 100,000.

Solution:

  2. 100,000 + 250,000 = 350,000

  4. 600,000 + 250,000 = 850,000
