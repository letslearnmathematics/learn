Introduction

Linear relations describe relationships between two variables that form a straight line when graphed. These graphs are powerful tools for solving real-life problems by visually representing relationships between quantities.

In this lesson, we'll learn how to draw and interpret graphs of linear relations to solve practical problems. You'll see how these graphs can help answer questions about costs, distances, speeds, and many other real-world situations.

Understanding Graphs

Key Components of a Graph

Every graph has essential components that help us interpret the information it represents:

Axes and Scales

The two perpendicular lines on a graph are called axes:

  • X-axis: The horizontal axis (usually represents the independent variable)
  • Y-axis: The vertical axis (usually represents the dependent variable)

Each axis must have a consistent scale with appropriate units.

Coordinates and Points

Points on a graph are represented by coordinates (x, y):

\[ \text{Example: } (2, 5) \]

This means 2 units along the x-axis and 5 units up the y-axis.

Plotting Graphs

Steps to Plot a Linear Graph

Follow these steps to accurately plot a linear graph from an equation or table of values:

From an Equation

To plot a graph from an equation like y = 2x + 1:

  1. Create a table of values (choose x values and calculate y)
  2. Plot the (x, y) points on graph paper
  3. Draw a straight line through the points

Example table for y = 2x + 1:

x -2 -1 0 1 2
y -3 -1 1 3 5

From Real-Life Data

When plotting from real-life data:

  1. Identify the two related quantities (variables)
  2. Decide which is independent (x) and dependent (y)
  3. Choose appropriate scales for both axes
  4. Plot the given data points
  5. Draw the line of best fit

Real Life Applications

Common Applications

Linear graphs are used in many real-life situations. Here are some common examples:

Cost Analysis

Graphs can show how costs change with quantity:

Example: A printing shop charges Gh₵5 setup fee plus Gh₵1 per page. The cost equation is:

\[ C = 5 + p \]

where C is total cost and p is number of pages.

The graph helps visualize how cost increases with each additional page.

Distance-Time Graphs

These show how distance changes over time:

Example: A car travels at constant speed of 60km/h. The distance equation is:

\[ d = 60t \]

where d is distance in km and t is time in hours.

The slope of the graph represents the speed.

Practice Exercise

Question 1
