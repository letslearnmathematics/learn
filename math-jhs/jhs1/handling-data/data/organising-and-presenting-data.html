Introduction

A questionnaire is a research instrument consisting of a series of questions for gathering information from respondents. In mathematics, we use questionnaires to collect data that we can then organize and analyze to answer questions.

Example of a simple questionnaire question:

How many people live in your household (including yourself)?

○ 1-2 people
○ 3-4 people
○ 5-6 people
○ More than 6 people

Types of Data

Data collected through questionnaires can be:

  • Numerical/Quantitative: Data that can be counted or measured (e.g., number of siblings, test scores)
  • Categorical/Qualitative: Data that can be grouped into categories (e.g., favorite color, mode of transportation to school)

Example: When collecting data about household sizes, we're collecting numerical data that we can count and organize.

Frequency Tables

Using Tallies to Organize Data

Tallies are marks (usually vertical lines) used to count items. Every fifth tally is drawn diagonally across the previous four to make counting easier.

Example 1: Mathematics Test Scores

Here are marks obtained by students in a mathematics test:

65, 72, 80, 65, 90, 72, 80, 65, 72, 90, 80, 65, 72, 80, 90

Mark Tally Frequency
65 |||| 4
72 |||| 4
80 |||| 4
90 ||| 3

Example 2: Household Sizes

Data collected about number of people living in households:

3, 5, 2, 4, 6, 3, 2, 4, 5, 3, 4, 2, 5, 6, 3, 4, 2, 5, 3, 4

Number of People Tally Frequency
2 ||| 3
3 |||| 5
4 |||| 5
5 |||| 4
6 || 2

Creating Frequency Tables

Steps to create a frequency table:

  1. List all possible values or ranges (for grouped data)
  2. Use tally marks to count occurrences of each value
  3. Count the tallies and record the frequency
  4. Add any additional columns needed (e.g., cumulative frequency)

Example: For the household size data above, we listed each possible value (2-6), tallied the occurrences, then counted the tallies to get frequencies.

Graphical Representation

Types of Graphs

Data from frequency tables can be represented graphically in different ways:

Bar Graph

Uses rectangular bars with lengths proportional to the values they represent.

Example: A bar graph for the household size data would have bars for each household size (2-6) with heights corresponding to their frequencies.

Pie Chart

A circular graph divided into sectors, where each sector represents a proportion of the whole.

Example: A pie chart for the test scores would have slices sized according to how many students got each mark.

Pictograph

Uses pictures or symbols to represent data values.

Example: A pictograph for household sizes might use small house icons, with each icon representing 1 household of that size.

Answering Questions from Graphs

Graphs help us answer questions about the data and identify patterns:

  1. Which category has the highest/lowest frequency?
  2. What is the difference between two categories?
  3. What is the total number of observations?
  4. What percentage does each category represent?

Example questions for household size data:

  • What is the most common household size?
  • How many households have more than 4 people?
  • What percentage of households have exactly 3 people?

Questionnaire Design

Creating Effective Questionnaires

A good questionnaire should:

  • Have clear, simple questions
  • Use appropriate question types (multiple choice, rating scales, open-ended)
  • Be logically organized
  • Be easy to complete
  • Collect data that can be easily analyzed

Example questionnaire about school transportation:

1. How do you usually travel to school?
○ Walk
○ Bicycle
○ Bus
○ Car
○ Other (please specify) ________

2. How long does your journey to school usually take?
○ Less than 10 minutes
○ 10-20 minutes
○ 21-30 minutes
○ More than 30 minutes

Administering Questionnaires

When administering a questionnaire:

  1. Explain the purpose clearly
  2. Ensure respondents understand the questions
  3. Maintain confidentiality
  4. Collect responses systematically
  5. Thank participants for their time

Question 4

If you created a bar graph for the siblings data, which number of siblings would have the tallest bar?

Both 1 sibling and 2 siblings have the highest frequency (6 each), so their bars would be equally tallest.

Question 5

What percentage of the households in the first example have exactly 4 people?

Total households = 3 + 5 + 5 + 4 + 2 = 19

Households with 4 people = 5

Percentage = (5 ÷ 19) × 100 ≈ 26.3%

Question 6

What questions could you ask about the test score data presented in a bar graph?

Possible questions:

  • Which test score was most common?
  • How many students scored 80 or above?
  • What was the difference in frequency between the highest and lowest scores?
  • What percentage of students scored below 80?
