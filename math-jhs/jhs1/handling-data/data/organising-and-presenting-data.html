Introduction

Data organization is the process of arranging collected information in a meaningful way so that it can be easily understood and analyzed. We often organize data into tables and charts to help us see patterns and draw conclusions.

Common ways to organize data:

  • Frequency tables
  • Bar charts
  • Pie charts
  • Line graphs

Understanding Data Collection

Before we can organize data, we need to collect it. Data can come from:

  • Surveys: Asking people questions
  • Measurements: Recording quantities or sizes
  • Observations: Noting what we see
  • Experiments: Testing ideas and recording results

Example: Surveying classmates about their favorite subjects

Frequency Tables

Using Tallies to Organize Data

Tallies are marks (usually vertical lines) used to count items quickly. Every fifth tally is drawn diagonally across the previous four to make groups of five.

Example 1: Test Scores

Here are marks obtained by students in a mathematics test:

85, 72, 90, 85, 78, 85, 72, 90, 78, 85, 72, 78, 90, 85, 72

Score Tally Frequency
72 |||| 4
78 ||| 3
85 |||| \ 5
90 ||| 3

Example 2: Household Survey

Number of people living in households:

3, 5, 4, 6, 3, 2, 4, 5, 6, 4, 3, 2, 5, 4, 3, 4, 5, 6, 4, 3

Number of People Tally Frequency
2 || 2
3 |||| \ 5
4 |||| \ 6
5 |||| 4
6 ||| 3

Creating a Frequency Table

Steps to create a frequency table:

  1. List all possible values or categories
  2. Use tally marks to count occurrences
  3. Add up the tallies to get frequencies
  4. Present in a clear table format

Example: Favorite colors of students

Color Tally Frequency
Red |||| 4
Blue |||| \ 5
Green ||| 3
Yellow || 2

Graphs and Charts

Types of Charts

We can present data from frequency tables in different types of charts:

Bar Charts

Use rectangular bars to show frequencies. The height of each bar represents the frequency.

Example: Bar chart of test scores

Pie Charts

Show proportions as slices of a circle. Each slice represents a category's percentage.

Example: Pie chart of favorite colors

Line Graphs

Show changes over time with connected points.

Example: Temperature changes over a week

Drawing a Bar Chart

Steps to draw a bar chart:

  1. Draw two perpendicular axes (x and y)
  2. Label the x-axis with categories
  3. Label the y-axis with frequencies
  4. Draw bars of appropriate heights
  5. Add a title and labels

Example: Bar chart for household survey data

Data Analysis

Answering Questions from Data

We can use organized data to answer questions and solve problems:

Example 1: Test Scores

From our test scores frequency table:

  • Which score appeared most frequently? (85 - mode)
  • How many students scored above 80? (5 + 3 = 8)
  • What percentage scored 78? (3/15 = 20%)

Example 2: Household Survey

From our household data:

  • What is the most common household size? (4 - mode)
  • How many households have more than 4 people? (4 + 3 = 7)
  • What fraction of households have 3 people? (5/20 = 1/4)

Posing Questions from Data

We can also create questions based on the data presented:

  1. Look for patterns in the data
  2. Identify highest and lowest values
  3. Calculate differences between categories
  4. Find proportions and percentages

Example questions for favorite colors data:

  • Which color is least popular?
  • How many more students prefer blue than red?
  • What percentage of students chose green?
  • If we surveyed 10 more students and 3 chose yellow, what would be the new frequency?

Question 3

From the frequency table in Question 2, answer: a) What is the most common number of children? b) How many families have more than 2 children? c) What percentage have exactly 1 child?
