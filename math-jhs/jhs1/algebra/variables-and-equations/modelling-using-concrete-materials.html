Introduction

Linear equations are mathematical statements that show two expressions are equal. They often contain variables (like x) that we need to solve for. We can model these equations using concrete materials like counters and integer tiles to help visualize and understand the solving process.

\[ \text{Example equation: } 2x + 3 = 7 \]

Understanding Equations

An equation is like a balance scale - both sides must remain equal. When we solve equations, we perform the same operation to both sides to maintain the balance:

  • Variables: Represent unknown values (often shown as shapes or letters)
  • Constants: Known values (numbers or counters)
  • Balance: Both sides of the equation must remain equal

Example: In \( x + 2 = 5 \), x is the variable, 2 and 5 are constants.

Objects on Balance

Using Counters

We can model equations using counters (like coins or blocks) where:

  1. Each counter represents +1
  2. Variables are represented by containers or unknown groups
  3. The balance scale shows both sides are equal

Example 1: \( x + 3 = 5 \)

Left side: 1 bag (x) + 3 counters

Right side: 5 counters

Remove 3 counters from both sides: x = 2

Example 2: \( 2x = 8 \)

Left side: 2 identical bags (each x)

Right side: 8 counters

Divide both sides by 2: x = 4

Solving Step-by-Step

The process for solving equations with objects:

  1. Represent the equation with objects on a balance scale
  2. Add/remove the same objects from both sides to keep balance
  3. Combine like terms by grouping identical objects
  4. Isolate the variable by removing other objects

Example: Solve \( x + 4 = 2x + 1 \)

1. Remove x from both sides: 4 = x + 1

2. Remove 1 from both sides: 3 = x

Shapes on Balance

Using Geometric Shapes

We can use shapes to represent variables in equations:

Squares as Variables

Each square represents one variable (x):

\[ \text{Example: } \square + 3 = 5 \]

Remove 3 from both sides: \(\square = 2\)

Different Shapes

Different shapes can represent different variables:

\[ \text{Example: } \triangle + \square = 5 \]

If \(\triangle = 2\), then \(\square = 3\)

Solving with Shapes

To solve equations with shapes:

  1. Identify what each shape represents
  2. Maintain balance by doing the same to both sides
  3. Substitute known values when possible
  4. Simplify by combining like shapes

Example: Solve \( 2\square + 1 = \square + 4 \)

1. Remove \(\square\) from both sides: \(\square + 1 = 4\)

2. Remove 1 from both sides: \(\square = 3\)

Symbolic Methods

Connecting to Algebra

The concrete models lead directly to symbolic algebraic methods:

Addition/Subtraction

Adding/removing objects becomes adding/subtracting terms:

\[ x + 3 = 7 \] \[ x + 3 - 3 = 7 - 3 \] \[ x = 4 \]

Multiplication/Division

Grouping objects becomes multiplying/dividing:

\[ 2x = 8 \] \[ \frac{2x}{2} = \frac{8}{2} \] \[ x = 4 \]

Solving Process

The general process for solving linear equations:

  1. Simplify both sides (distribute, combine like terms)
  2. Move variable terms to one side, constants to the other
  3. Isolate the variable by multiplying/dividing
  4. Check your solution by substituting back in

Example: Solve \( 2(x + 1) = 10 \)

1. Distribute: \( 2x + 2 = 10 \)

2. Subtract 2: \( 2x = 8 \)

3. Divide by 2: \( x = 4 \)

Practice Exercise

Question 1

Model and solve \( x + 2 = 5 \) using counters

Left side: 1 bag + 2 counters

Right side: 5 counters

Remove 2 counters from both sides: x = 3

Question 2

Solve \( 3\square = 12 \) using shapes

Divide both sides by 3: \(\square = 4\)

Question 3

Solve \( 2x + 1 = x + 4 \) symbolically

Subtract x: \( x + 1 = 4 \)

Subtract 1: \( x = 3 \)

Question 4

Model \( 2y = y + 3 \) with shapes and solve

Left side: 2 triangles

Right side: 1 triangle + 3 counters

Remove 1 triangle: \( \triangle = 3 \)

Question 5

Solve \( 4 = 5 - x \) using counters

Right side: 5 counters minus 1 bag

Add x to both sides: \( 4 + x = 5 \)

Subtract 4: \( x = 1 \)

Question 6

Solve \( 2(\square + 1) = 8 \) using shapes

Divide both sides by 2: \( \square + 1 = 4 \)

Subtract 1: \( \square = 3 \)
