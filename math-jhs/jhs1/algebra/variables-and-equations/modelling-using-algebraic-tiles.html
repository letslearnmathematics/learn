Introduction

A linear equation is an equation that makes a straight line when graphed. It typically has one or two variables, usually x and y, with no exponents or powers. In this lesson, we'll learn how to model these equations using algebraic tiles and solve them step by step.

\[ \text{Examples: } 2x + 3 = 7, \quad y = 4x - 2 \]

Understanding Linear Equations

A linear equation has three main components:

  • Variables: Letters that represent unknown values (usually x or y)
  • Coefficients: Numbers multiplying the variables (like the 2 in 2x)
  • Constants: Fixed numbers in the equation (like the 3 in 2x + 3 = 7)

Example: In \(3x - 5 = 10\):

  • Variable: x
  • Coefficient: 3
  • Constants: -5 and 10

Algebraic Tiles

Understanding Algebraic Tiles

Algebraic tiles are physical or visual tools that help us represent and solve equations. There are three main types:

  1. Unit tiles: Small squares representing +1 or -1
  2. Variable tiles: Rectangles representing +x or -x
  3. Variable squared tiles: Large squares representing x² or -x²

Dealing with Negative Values

When you have negative tiles, you can add zero pairs to help solve the equation:

Example: Solve \(3x - 2 = 4\)

  1. Model the equation (3x tiles and 2 negative unit tiles on left, 4 positive units on right)
  2. Add 2 positive units to both sides (creating zero pairs with the negatives)
  3. This gives \(3x = 6\)
  4. Divide both sides by 3 to get \(x = 2\)

Practice Exercise

Question 1

Model and solve \(x + 4 = 7\) using algebraic tiles

1. Model: x tile + 4 unit tiles = 7 unit tiles

2. Remove 4 units from both sides: x = 3

Solution: \(x = 3\)

Question 2

Solve \(2x - 3 = 5\) using tiles

1. Model: 2x tiles + 3 negative units = 5 positive units

2. Add 3 units to both sides: 2x = 8

3. Divide both sides by 2: x = 4

Question 3

What is a zero pair in algebra tiles?

A positive and negative tile of the same type that cancel each other out (like +1 and -1, or +x and -x).

Question 4

Model and solve \(5 = 2x + 1\)

1. Model: 5 unit tiles = 2x tiles + 1 unit tile

2. Remove 1 unit from both sides: 4 = 2x

3. Divide both sides by 2: x = 2

Question 5

Solve \(4x + 2 = 3x + 5\) using tiles

1. Model both sides with tiles

2. Remove 3x tiles from both sides: x + 2 = 5

3. Remove 2 units from both sides: x = 3

Question 6

Why must you do the same operation to both sides of an equation?

To maintain the balance of the equation. An equation is like a balanced scale - whatever you do to one side must be done to the other to keep it balanced.
