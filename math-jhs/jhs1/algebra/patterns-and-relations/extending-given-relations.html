Introduction

Extending given relations involves recognizing patterns in numbers or symbols and continuing them according to established rules. This skill helps us predict future terms in sequences and understand mathematical relationships.

\[ \text{Example: } 2, 4, 6, 8, \underline{\quad} \]

Key Concepts

A relation shows how numbers or objects are connected:

  • Numerical Relations: Patterns in number sequences
  • Symbolic Relations: Relationships expressed with variables
  • Tabular Relations: Patterns shown in tables

Example: In the relation y = x + 2, when x is 3, y is 5.

Extending Symbolic Relations

Understanding Symbolic Relations

A symbolic relation uses variables to show how quantities are related. We can extend these relations by substituting values.

Example 1: y = 2x + 1

Example 2: a = b × 3 - 2

Extending Simple Equations

To extend a given relation:

  1. Identify the pattern or rule
  2. Apply the rule to new values
  3. Verify the pattern continues

Given: m = n + 5

When n = 10, m = 15

When n = 15, m = 20

Number Patterns and Tables

Recognizing Number Patterns

Number patterns follow specific rules that can be identified and extended:

Arithmetic Sequences

Add or subtract the same number each time:

\[ 5, 8, 11, 14, \underline{17} \text{ (Rule: Add 3)} \]

Geometric Sequences

Multiply or divide by the same number each time:

\[ 3, 6, 12, 24, \underline{48} \text{ (Rule: Multiply by 2)} \]

Special Patterns

Other patterns like square numbers or alternating sequences:

\[ 1, 4, 9, 16, \underline{25} \text{ (Square numbers)} \]

Completing Tables

To complete a table of values:

  1. Study the given values to find the pattern
  2. Apply the pattern to missing values
  3. Check your answers make sense

Complete the table:

Input (x) Output (y)
1 4
2 7
3 10
4 13

Rule: y = 3x + 1

Finding the Rule

To find the rule in a pattern:

  1. Look at how numbers change from one term to the next
  2. Check if it's addition, subtraction, multiplication, or division
  3. Test your rule on known values

Given sequence: 2, 5, 8, 11, ...

Pattern: +3 each time

Next term: 14

Applications

Practical Examples

Extending relations is useful in many real-world situations:

Savings Growth

If you save $10 each week, how much after 8 weeks?

\[ \text{Week 1: \$10, Week 2: \$20, ..., Week 8: \$80} \]

Plant Growth

A plant grows 2cm each day. How tall after 1 week?

\[ \text{Day 1: 2cm, Day 2: 4cm, ..., Day 7: 14cm} \]

Pricing

If 3 pencils cost \$1.50, how much for 7 pencils?

\[ \text{Each pencil: \$0.50} \] \[ \text{7 pencils: \$3.50} \]
