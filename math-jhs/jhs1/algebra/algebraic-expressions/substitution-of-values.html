Introduction

Algebraic expressions are mathematical phrases that can contain numbers, variables (like x or y), and operations (like +, -, ×, ÷). Evaluating an expression means finding its value by substituting numbers for the variables.

\[ \text{Example expressions: } 3x + 2, \quad 5y - 7, \quad 2a^2 + 3b - 5 \]

Understanding Variables

Variables are symbols (usually letters) that represent unknown values. When we evaluate expressions, we replace these variables with specific numbers.

  • Variables: Letters like x, y, a, b that represent unknown values
  • Constants: Fixed numbers in the expression (like the 2 in 3x + 2)
  • Coefficients: Numbers multiplying variables (like the 3 in 3x)

Example: In \(3x + 2\), x is the variable, 3 is the coefficient, and 2 is the constant.

Simplifying Expressions

Combining Like Terms

Before substituting values, we can often simplify expressions by combining like terms (terms with the same variable raised to the same power):

  1. Identify terms with the same variable and exponent
  2. Add or subtract their coefficients
  3. Keep the variable part unchanged

Example 1: Simplify \(2x + 3x - x\)

\[ (2 + 3 - 1)x = 4x \]

Example 2: Simplify \(5y - 2y + 7 - 3\)

\[ (5 - 2)y + (7 - 3) = 3y + 4 \]

Using the Distributive Property

The distributive property lets us remove parentheses by multiplying the term outside with each term inside:

\begin{eqnarray*} a(b + c) &=& ab + ac \\ 3(x + 4) &=& 3x + 12 \\ 2(3y - 5) &=& 6y - 10 \end{eqnarray*}

Substituting Values

Step-by-Step Evaluation

To evaluate an expression by substitution:

  1. Replace each variable with its given value
  2. Follow the order of operations (PEMDAS/BODMAS)
  3. Simplify to find the final value

Example 1: Evaluate \(2x + 3\) when \(x = 5\)

\[ 2(5) + 3 = 10 + 3 = 13 \]

Example 2: Evaluate \(3y^2 - 2y + 1\) when \(y = 4\)

\[ 3(4)^2 - 2(4) + 1 = 3(16) - 8 + 1 = 48 - 8 + 1 = 41 \]

Multiple Variables

When expressions have multiple variables, substitute each one carefully:

Evaluate \(2a + 3b - c\) when \(a = 4\), \(b = 2\), \(c = 5\):

\[ 2(4) + 3(2) - 5 = 8 + 6 - 5 = 9 \]

Common Mistakes to Avoid

  • Forgetting to multiply coefficients with variables
  • Miscounting negative signs when substituting
  • Applying exponents to the wrong terms
  • Not following the correct order of operations

Incorrect: Evaluating \(3x^2\) as \((3x)^2\) when \(x = 2\)

\[ \text{Correct: } 3(2)^2 = 12 \] \[ \text{Incorrect: } (3 \times 2)^2 = 36 \]

Applications

Practical Examples

Evaluating expressions is useful in many real-world situations:

Calculating Costs

If items cost \$x each and you buy 5, total cost is \(5x\):

\[ \text{When } x = 12: 5(12) = \$60 \]

Temperature Conversion

Convert Fahrenheit to Celsius: \(C = \frac{5}{9}(F - 32)\)

\[ \text{When } F = 77: \frac{5}{9}(77 - 32) = 25°C \]

Geometry Formulas

Area of rectangle: \(A = lw\)

\[ \text{When } l = 8, w = 5: A = 8 \times 5 = 40 \text{ units}^2 \]

Practice Exercise

Question 1

Evaluate \(4x - 7\) when \(x = 3\)

\[ 4(3) - 7 = 12 - 7 = 5 \]

Question 2

Evaluate \(2y^2 + 3y - 1\) when \(y = 2\)

\[ 2(2)^2 + 3(2) - 1 = 8 + 6 - 1 = 13 \]

Question 3

Evaluate \(5a - 2b\) when \(a = 4\), \(b = 3\)

\[ 5(4) - 2(3) = 20 - 6 = 14 \]

Question 4

Evaluate \(\frac{x + 5}{2}\) when \(x = 7\)

\[ \frac{7 + 5}{2} = \frac{12}{2} = 6 \]

Question 5

Evaluate \(3(x + 4) - 2x\) when \(x = 5\)

\[ 3(5 + 4) - 2(5) = 27 - 10 = 17 \]

Question 6

Evaluate \(2m^2 - mn + n^2\) when \(m = 3\), \(n = 2\)

\[ 2(3)^2 - (3)(2) + (2)^2 = 18 - 6 + 4 = 16 \]
