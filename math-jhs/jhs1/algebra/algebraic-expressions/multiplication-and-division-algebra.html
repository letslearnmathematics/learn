Introduction

Algebraic expressions are mathematical phrases that can contain numbers, variables, and operation symbols. In this lesson, we'll learn how to multiply and divide algebraic expressions with rational coefficients (fractions).

\[ \text{Examples: } 3x, \frac{1}{2}y, 4a^2b, \frac{2x}{3} \]

Key Terms

  • Term: A single part of an algebraic expression (e.g., 3x, 5y²)
  • Coefficient: The numerical factor of a term (e.g., in 4x, 4 is the coefficient)
  • Variable: A symbol (usually a letter) that represents an unknown value
  • Rational coefficient: A coefficient that is a fraction (e.g., ½x)

Example: In \(\frac{3}{4}ab^2\), \(\frac{3}{4}\) is the coefficient, \(a\) and \(b\) are variables

Multiplication of Algebraic Expressions

Multiplying Terms

To multiply algebraic terms:

  1. Multiply the coefficients (including fractions)
  2. Multiply the variables by adding exponents of like bases

Example 1: Multiply \(\frac{1}{2}x\) and \(\frac{3}{4}y\)

\[ \frac{1}{2}x \times \frac{3}{4}y = \left(\frac{1}{2} \times \frac{3}{4}\right) \times (x \times y) = \frac{3}{8}xy \]

Example 2: Multiply \(2a\) and \(\frac{1}{3}a^2b\)

\[ 2a \times \frac{1}{3}a^2b = \left(2 \times \frac{1}{3}\right) \times (a \times a^2 \times b) = \frac{2}{3}a^3b \]

Multiplying Expressions

To multiply expressions with multiple terms, use the distributive property:

Example 1: \(\frac{1}{2}(4x + 6y)\)

\[ = \frac{1}{2} \times 4x + \frac{1}{2} \times 6y = 2x + 3y \]

Example 2: \((3x + 2y)\left(\frac{1}{3}x\right)\)

\[ = 3x \times \frac{1}{3}x + 2y \times \frac{1}{3}x = x^2 + \frac{2}{3}xy \]

Division of Algebraic Expressions

Dividing Terms

To divide algebraic terms:

  1. Divide the coefficients (multiply by reciprocal for fractions)
  2. Divide the variables by subtracting exponents of like bases

Example 1: Divide \(\frac{3}{4}x\) by \(\frac{1}{2}\)

\[ \frac{\frac{3}{4}x}{\frac{1}{2}} = \frac{3}{4}x \times \frac{2}{1} = \frac{6}{4}x = \frac{3}{2}x \]

Example 2: Divide \(6a^2b\) by \(\frac{2}{3}ab\)

\[ \frac{6a^2b}{\frac{2}{3}ab} = 6a^2b \times \frac{3}{2a^1b^1} = \frac{18}{2}a^{2-1}b^{1-1} = 9a \]

Dividing Expressions

To divide expressions with multiple terms, divide each term separately:

Example: \(\frac{4x + 8y}{\frac{1}{2}}\)

\[ = \frac{4x}{\frac{1}{2}} + \frac{8y}{\frac{1}{2}} = 4x \times 2 + 8y \times 2 = 8x + 16y \]

Example: \(\frac{9a^2b - 6ab^2}{3ab}\)

\[ = \frac{9a^2b}{3ab} - \frac{6ab^2}{3ab} = 3a - 2b \]

Applications

Practical Examples

Algebraic expressions are used in many real-world situations:

Area Calculations

Find the area of a rectangle with length \(\frac{3}{4}x\) and width \(\frac{2}{5}y\):

\[ \text{Area} = \frac{3}{4}x \times \frac{2}{5}y = \frac{6}{20}xy = \frac{3}{10}xy \]

Speed Problems

If distance = \(\frac{5}{2}x\) km and time = \(\frac{1}{4}x\) hours, find speed:

\[ \text{Speed} = \frac{\frac{5}{2}x}{\frac{1}{4}x} = \frac{5}{2} \times \frac{4}{1} = 10 \text{ km/h} \]

Recipe Scaling

If a recipe calls for \(\frac{2}{3}a\) cups of flour and you're making \(\frac{1}{2}\) of the recipe:

\[ \frac{2}{3}a \times \frac{1}{2} = \frac{2}{6}a = \frac{1}{3}a \text{ cups} \]

Practice Exercise

Question 1

Multiply \(\frac{2}{3}a\) and \(\frac{3}{4}b\)

\[ \frac{2}{3}a \times \frac{3}{4}b = \frac{6}{12}ab = \frac{1}{2}ab \]

Question 2

Divide \(\frac{5}{6}x^2\) by \(\frac{1}{3}x\)

\[ \frac{\frac{5}{6}x^2}{\frac{1}{3}x} = \frac{5}{6}x^2 \times \frac{3}{1x} = \frac{15}{6}x = \frac{5}{2}x \]

Question 3

Simplify \(\frac{1}{2}(4x + 8y)\)

\[ \frac{1}{2} \times 4x + \frac{1}{2} \times 8y = 2x + 4y \]

Question 4

Divide \((9a^2b - 3ab^2)\) by \(3ab\)

\[ \frac{9a^2b}{3ab} - \frac{3ab^2}{3ab} = 3a - b \]

Question 5

Multiply \((2x + 3y)\) by \(\frac{1}{4}\)

\[ 2x \times \frac{1}{4} + 3y \times \frac{1}{4} = \frac{1}{2}x + \frac{3}{4}y \]

Question 6

Divide \(\frac{7}{8}xy\) by \(\frac{1}{2}x\)

\[ \frac{\frac{7}{8}xy}{\frac{1}{2}x} = \frac{7}{8}xy \times \frac{2}{1x} = \frac{14}{8}y = \frac{7}{4}y \]
