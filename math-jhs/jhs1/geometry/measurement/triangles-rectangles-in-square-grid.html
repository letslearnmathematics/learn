Introduction

A square grid is made up of equal squares arranged in rows and columns. We can use these grids to accurately draw rectangles and triangles with specific dimensions.

Each small square in the grid represents 1 unit by 1 unit.

Understanding Dimensions

When we talk about dimensions of shapes on a grid:

  • Length: How many squares along the bottom (horizontal)
  • Width/Height: How many squares up the side (vertical)

Example: A rectangle with dimensions 5 units by 3 units means:

  • 5 squares long (left to right)
  • 3 squares tall (bottom to top)

Drawing Rectangles

Steps to Draw a Rectangle

To draw a rectangle with given dimensions on a grid:

  1. Identify the length and width from the given dimensions
  2. Choose a starting point on the grid
  3. Count the number of squares for the length to the right
  4. From that point, count up the number of squares for the height
  5. Complete the rectangle by connecting all sides

Example 1: Drawing a 4×2 rectangle

4 units long (horizontal) and 2 units tall (vertical)

Example 2: Drawing a 3×5 rectangle

3 units long (horizontal) and 5 units tall (vertical)

Special Rectangles

Some rectangles have special properties:

Square: A rectangle with all sides equal (e.g., 3×3, 5×5)

Oblong: A rectangle with unequal sides (e.g., 2×4, 3×6)

Drawing Triangles

Types of Triangles on Grids

We can draw different types of triangles on square grids:

Right-Angled Triangles

Has one 90° angle, with sides meeting at right angles

Example: Base 4 units, height 3 units

Isosceles Triangles

Has two sides of equal length

Example: Two sides of 5 units, base of 6 units

Scalene Triangles

All sides have different lengths

Example: Sides of 3, 4, and 5 units

Steps to Draw a Triangle

To draw a triangle with given dimensions on a grid:

  1. Identify the base length and height
  2. Draw the base line using the given number of units
  3. From one end of the base, count up the height
  4. Connect this point to the other end of the base
  5. For specific triangles, ensure sides meet at correct angles
