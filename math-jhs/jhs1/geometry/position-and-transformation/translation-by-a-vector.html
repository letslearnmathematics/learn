Introduction

Translation is a type of transformation that moves every point of a shape the same distance in the same direction. It's like sliding an object without rotating or flipping it.

Real-world example: When people go down a slide, their position changes through translation.

Understanding Coordinates

A coordinate plane has two axes:

  • x-axis: The horizontal axis (left-right)
  • y-axis: The vertical axis (up-down)

Points are written as (x, y). Example: (3, 2) means 3 units right and 2 units up.

Plotting Points & Shapes

Plotting Points

To plot a point on the coordinate plane:

  1. Start at the origin (0,0)
  2. Move horizontally according to the x-coordinate
  3. Move vertically according to the y-coordinate
  4. Mark the point

Example 1: Plot (2, 3)

From (0,0), move 2 units right and 3 units up

Example 2: Plot (-1, 4)

From (0,0), move 1 unit left and 4 units up

Plotting Shapes

To plot a shape:

  1. Plot all the vertices (corners) of the shape
  2. Connect the points in order
  3. Close the shape by connecting back to the first point

Example: Plot triangle ABC with vertices A(1,1), B(3,1), C(2,3)

Translation Vectors

Understanding Vectors

A translation vector tells us how far and in what direction to move a shape. It's written as \(\begin{pmatrix} x \\ y \end{pmatrix}\) where:

  • x = horizontal movement (right if positive, left if negative)
  • y = vertical movement (up if positive, down if negative)

Example: \(\begin{pmatrix} 3 \\ -2 \end{pmatrix}\) means move 3 units right and 2 units down

Translating Points

To translate a point using a vector:

  1. Add the x-component of the vector to the x-coordinate
  2. Add the y-component of the vector to the y-coordinate

Example: Translate point (2,5) by vector \(\begin{pmatrix} -1 \\ 3 \end{pmatrix}\):

\[ (2 + (-1), 5 + 3) = (1, 8) \]

Translating Shapes

To translate a shape, translate all its vertices by the same vector:

Example: Translate triangle ABC with vertices A(1,1), B(3,1), C(2,3) by \(\begin{pmatrix} 2 \\ -1 \end{pmatrix}\):

New vertices:

  • A' = (1+2, 1-1) = (3,0)
  • B' = (3+2, 1-1) = (5,0)
  • C' = (2+2, 3-1) = (4,2)

Applications

Practical Examples

Translations are used in many real-world situations:

Slide Movement

As people go down a slide, their position changes through translation.

Computer Graphics

Moving objects in video games uses translation vectors.

Architecture

Copying design elements to different positions in a blueprint.

Practice Exercise

Question 1

Plot the point (4, -2) on a coordinate plane.

From (0,0), move 4 units right and 2 units down.

Question 2

Translate point (3, 5) by vector \(\begin{pmatrix} -2 \\ 4 \end{pmatrix}\). What is the new point?

New point: (3 + (-2), (5 + 4) = (1, 9)

Question 3

A triangle has vertices at (1,1), (2,4), and (3,1). Translate it by \(\begin{pmatrix} 3 \\ -2 \end{pmatrix}\). Find the new vertices.

New vertices:

  • (1+3, 1-2) = (4, -1)
  • (2+3, 4-2) = (5, 2)
  • (3+3, 1-2) = (6, -1)

Question 4

Describe the translation vector that moves (2,3) to (5,0).

Vector: \(\begin{pmatrix} 5-2 \\ 0-3 \end{pmatrix} = \begin{pmatrix} 3 \\ -3 \end{pmatrix}\)

Question 5

Plot a square with vertices (1,1), (3,1), (3,3), (1,3). Then translate it by \(\begin{pmatrix} -2 \\ 1 \end{pmatrix}\).

New vertices:

  • (1-2, 1+1) = (-1, 2)
  • (3-2, 1+1) = (1, 2)
  • (3-2, 3+1) = (1, 4)
  • (1-2, 3+1) = (-1, 4)

Question 6

If a point moves from (4,2) to (1,5), what is the translation vector?

Vector: \(\begin{pmatrix} 1-4 \\ 5-2 \end{pmatrix} = \begin{pmatrix} -3 \\ 3 \end{pmatrix}\)
