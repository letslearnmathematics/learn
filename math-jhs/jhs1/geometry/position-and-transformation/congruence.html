Introduction

In geometry, shapes can be congruent or similar. These concepts help us understand how shapes relate to each other in size and shape.

Congruent shapes are identical in size and shape.

Similar shapes have the same shape but different sizes.

Key Differences

Property Congruent Shapes Similar Shapes
Size Same Different
Shape Same Same
Angles Equal Equal
Sides Equal length Proportional length

Congruent Shapes

Properties of Congruent Shapes

Two shapes are congruent if:

  1. All corresponding sides are equal in length
  2. All corresponding angles are equal in measure
  3. They have exactly the same size and shape

Example 1: Triangles

Triangle ABC ≅ Triangle DEF means:

  • AB = DE, BC = EF, AC = DF
  • ∠A = ∠D, ∠B = ∠E, ∠C = ∠F

Example 2: Rectangles

Rectangle PQRS ≅ Rectangle WXYZ means:

  • PQ = WX, QR = XY, RS = YZ, SP = ZW
  • All angles are 90° in both

Verifying Congruence

To verify if shapes are congruent in the coordinate plane:

  1. Calculate lengths of all sides using distance formula
  2. Measure all angles (or verify they match based on side lengths)
  3. Check if corresponding parts are equal

Distance Formula: \( d = \sqrt{(x_2-x_1)^2 + (y_2-y_1)^2} \)

Example: Verify if triangles with vertices A(1,2), B(4,2), C(1,5) and D(3,6), E(3,3), F(6,6) are congruent.

Similar Shapes

Properties of Similar Shapes

Two shapes are similar if:

  1. All corresponding angles are equal
  2. All corresponding sides are proportional
  3. They have the same shape but different sizes

Example 1: Triangles

Triangle ABC ~ Triangle DEF means:

  • ∠A = ∠D, ∠B = ∠E, ∠C = ∠F
  • AB/DE = BC/EF = AC/DF

Example 2: Polygons

For similar polygons:

  • All angles match
  • Side lengths have constant ratio
  • Area ratio is square of length ratio

Verifying Similarity

To verify similarity in the coordinate plane:

  1. Check if corresponding angles are equal
  2. Calculate side lengths and find ratios
  3. Verify all corresponding sides have the same ratio

Example: Verify if rectangles with vertices A(1,1), B(5,1), C(5,3), D(1,3) and E(2,4), F(6,4), G(6,8), H(2,8) are similar.

Real-life Applications

Everyday Examples

Congruent and similar shapes appear in many real-world situations:

Carpet Designs

Repeating patterns often use congruent shapes:

  • Identical motif shapes repeated
  • May be rotated or reflected but same size

Fabric Patterns

Similar shapes in textile designs:

  • Same flower pattern in different sizes
  • Proportional scaling maintains shape

Architecture

Building designs use both concepts:

  • Identical windows (congruent)
  • Scaled versions of decorative elements (similar)

Practical Verification

How to verify congruence/similarity in real objects:

  1. Measurement: Use rulers/protractors to check lengths and angles
  2. Comparison: Overlay shapes to check matching
  3. Scaling: Check if one is proportional enlargement of the other

Practice Exercise

Question 1

Which shapes are congruent in this set? (Provide image reference)

Shapes A and C are congruent because all corresponding sides and angles are equal.

Question 2

Are these two triangles similar? Triangle 1: sides 3,4,5; Triangle 2: sides 6,8,10

Yes, they are similar because all corresponding sides have ratio 1:2 and angles are equal.

Question 3

Verify if rectangles with vertices at (1,1), (4,1), (4,3), (1,3) and (2,2), (8,2), (8,6), (2,6) are similar.

First rectangle: width=3, height=2. Second rectangle: width=6, height=4. Ratio is 1:2 for both dimensions, so they are similar.

Question 4

Find real-world examples of (a) congruent shapes and (b) similar shapes in your classroom.

Possible answers:

(a) Congruent: identical floor tiles, same-sized windows

(b) Similar: different-sized circles on wall clock, scaled posters

Question 5

Are all congruent shapes similar? Are all similar shapes congruent? Explain.

All congruent shapes are similar (same shape AND size implies same shape).

Not all similar shapes are congruent (they may be different sizes).

Question 6

In a carpet design with repeating octagons, are the octagons congruent or similar?

If they are identical in size and shape, they are congruent. If they vary in size but keep the same shape, they are similar.
