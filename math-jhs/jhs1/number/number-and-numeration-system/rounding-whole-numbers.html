Introduction

When we round off, up or down a number, the new number is no more the actual value but an estimate or an approximated value. This is usually done in big businesses to make calculations easier for professionals like bankers.

We will be making use of this symbol \(\left(\approx\right)\), which is read as "approximately".

Whole Numbers \((\mathbb{W})\):

A whole number is a non-negative integer, meaning it incldeds all the numbers from 0, 1, 2, 3, \(\ldots\) and so on, without fractions or decimals.

The set of whole numbers is written as:
\[\mathbf{W} = \{0, 1, 2, 3, 4, \ldots\}\]

Remember

when rounding off, up or down a number, you need to consider whether the number to the immediate right of the place value you are rounding is part of any of the following sets of numbers:

Small Digits

\[ \{0, 1, 2, 3, 4\} \]

Big digits

\[ \{5, 6, 7, 8, 9\} \]

Rounding to Significant Figures

Video Lesson

Watch this comprehensive explanation of the lesson with worked examples.

Main Idea

Significant figures (or significant digits) refer to the digits in a number that carry meaning or contribute to the number's precision. These include all non-zero digits, any zeros between significant digits, and trailing zeros in the decimal part.

Rules

Rules for identifying significant figures:

1. Non-zero digits are always significant.
Example: 123 has three significant figures.

2. Any zeros between non-zero digits are significant.
Example: 1003 has four significant figures.

3. Trailing zeros in a number with a decimal point are significant.
Example: 100.0 has four significant figures.

4. Trailing zeros in a number without a decimal point are not significant.
Example: 100 has one significant figure.

5. Leading zeros (zeros before any non-zero digit) are not significant.
Example: 00234 has three significant figures.
