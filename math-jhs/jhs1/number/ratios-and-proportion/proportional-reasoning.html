Introduction

Proportional reasoning involves understanding relationships between quantities that change at the same rate. When two quantities are proportional, their ratio remains constant.

\[ \text{If } \frac{a}{b} = \frac{c}{d}, \text{ then } a \text{ and } b \text{ are proportional to } c \text{ and } d \]

Understanding Ratios

A ratio compares two quantities. When two ratios are equal, they form a proportion:

\[ \frac{2}{3} = \frac{4}{6} \text{ is a proportion because both ratios equal } \frac{2}{3} \]
  • Direct Proportion: As one quantity increases, the other increases at the same rate
  • Constant Ratio: The ratio between the two quantities always stays the same

Example: If 3 pencils cost \$1.50, then 6 pencils cost \$3.00 (same ratio of 2 pencils per \$1)

Finding Missing Values

Using Proportional Reasoning

To find missing values in proportional tables:

  1. Identify the constant ratio between known pairs
  2. Apply this ratio to find the missing value
  3. Check that all ratios in the table are equal

Example 1: Complete the table

Boxes Apples
2 24
5 ?

Solution: Find apples per box (24 ÷ 2 = 12 apples/box)

Then 5 boxes × 12 apples/box = 60 apples

Example 2: Using cross-multiplication

\[ \frac{3}{4} = \frac{x}{20} \] \[ 4x = 3 \times 20 \] \[ x = \frac{60}{4} = 15 \]

Proportional Tables

Proportional tables show equivalent ratios. All columns should have the same ratio when you divide corresponding values:

Time (hours) Distance (km) Ratio (km/h)
1 60 60
2 120 60
3 180 60
? 300 60

Missing value: 300 ÷ 60 = 5 hours

Coordinate Plane

Plotting Points

The coordinate plane has two perpendicular number lines (axes):

  • x-axis: Horizontal axis (left-right)
  • y-axis: Vertical axis (up-down)
  • Origin: (0,0) where axes intersect

Points are written as (x, y) pairs:

(3, 4) means 3 units right, 4 units up

(-2, 5) means 2 units left, 5 units up

Graphing Proportional Relationships

Proportional relationships graph as straight lines through the origin:

  1. Create a table of (x, y) values
  2. Plot each point on the coordinate plane
  3. Connect the points with a straight line

Example: y = 2x

x y
0 0
1 2
2 4

This would graph as a straight line through (0,0), (1,2), (2,4)

Identifying Proportions from Graphs

To determine if a graph shows a proportional relationship:

  • The line must be straight
  • It must pass through the origin (0,0)
  • The ratio y/x must be constant for all points

Example: These points show a proportion because:

3/1 = 6/2 = 9/3 = 3 (constant ratio)

And the line would pass through (0,0)

Applications

Practical Examples

Proportional reasoning is used in many real-world situations:

Recipe Scaling

If a cookie recipe for 24 cookies uses 2 cups of flour:

\[ \frac{2 \text{ cups}}{24 \text{ cookies}} = \frac{x}{36 \text{ cookies}} \] \[ x = 3 \text{ cups} \]

Map Scales

If 1 cm on a map equals 5 km in real life:

\[ \frac{1 \text{ cm}}{5 \text{ km}} = \frac{4 \text{ cm}}{x} \] \[ x = 20 \text{ km} \]

Speed Calculations

If a car travels 240 km in 3 hours at constant speed:

\[ \frac{240 \text{ km}}{3 \text{ hours}} = \frac{x}{5 \text{ hours}} \] \[ x = 400 \text{ km} \]

Practice Exercise

Question 1

Complete the table if y is proportional to x:

x y
2 10
5 ?

Ratio is 10/2 = 5, so y = 5x

For x=5: y=5×5=25

Question 2

Solve for x: \(\frac{7}{8} = \frac{x}{32}\)

\[ 8x = 7 \times 32 \] \[ 8x = 224 \] \[ x = 28 \]

Question 3

Plot the points (1,3), (2,6), (3,9) and determine if they show a proportional relationship

Yes, because:

3/1 = 6/2 = 9/3 = 3 (constant ratio)

And the line would pass through (0,0)

Question 4

If 5 notebooks cost \$12.50, how much would 8 notebooks cost?

\[ \frac{12.50}{5} = 2.50 \text{ per notebook} \] \[ 8 \times 2.50 = \$20.00 \]

Question 5

Which of these tables shows a proportional relationship?

Table A Table B
x:1, y:2 x:1, y:3
x:2, y:3 x:2, y:6
x:3, y:4 x:3, y:9

Table B shows a proportional relationship because:

3/1 = 6/2 = 9/3 = 3 (constant ratio)

Question 6

Graph the points (0,0), (1,4), (2,8) and write the proportional relationship as an equation

The equation is y = 4x

Because 4/1 = 8/2 = 4 (constant ratio)
