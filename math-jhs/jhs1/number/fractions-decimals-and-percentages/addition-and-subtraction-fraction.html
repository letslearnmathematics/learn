Introduction

Fractions represent parts of a whole. To add or subtract fractions, they must have the same denominator (the bottom number). When denominators are different, we find a common denominator before performing the operation.

\[ \text{Example: } \frac{1}{4} + \frac{2}{4} = \frac{3}{4} \]

Key Concepts

  • Numerator: The top number in a fraction (how many parts you have)
  • Denominator: The bottom number (how many equal parts the whole is divided into)
  • Like Fractions: Fractions with the same denominator
  • Unlike Fractions: Fractions with different denominators
  • Mixed Number: A whole number and a proper fraction combined (e.g., 2Â½)

Adding Fractions

Adding Like Fractions

When fractions have the same denominator, simply add the numerators and keep the denominator:

\[ \frac{a}{c} + \frac{b}{c} = \frac{a + b}{c} \]

Example 1:

\[ \frac{2}{5} + \frac{1}{5} = \frac{2 + 1}{5} = \frac{3}{5} \]

Example 2:

\[ \frac{3}{8} + \frac{2}{8} = \frac{5}{8} \]

Adding Unlike Fractions

When denominators are different:

  1. Find the Least Common Denominator (LCD)
  2. Convert each fraction to an equivalent fraction with the LCD
  3. Add the numerators
  4. Simplify if possible

Example:

\[ \frac{1}{2} + \frac{1}{3} \]

LCD of 2 and 3 is 6:

\[ \frac{1}{2} = \frac{3}{6}, \frac{1}{3} = \frac{2}{6} \] \[ \frac{3}{6} + \frac{2}{6} = \frac{5}{6} \]

Subtracting Fractions

Subtracting Like Fractions

When fractions have the same denominator, subtract the numerators and keep the denominator:

\[ \frac{a}{c} - \frac{b}{c} = \frac{a - b}{c} \]

Example 1:

\[ \frac{7}{8} - \frac{3}{8} = \frac{4}{8} = \frac{1}{2} \]

Subtracting Unlike Fractions

When denominators are different:

  1. Find the Least Common Denominator (LCD)
  2. Convert each fraction to an equivalent fraction with the LCD
  3. Subtract the numerators
  4. Simplify if possible

Example:

\[ \frac{3}{4} - \frac{1}{3} \]

LCD of 4 and 3 is 12:

\[ \frac{3}{4} = \frac{9}{12}, \frac{1}{3} = \frac{4}{12} \] \[ \frac{9}{12} - \frac{4}{12} = \frac{5}{12} \]

Mixed Numbers

Adding Mixed Numbers

To add mixed numbers:

  1. Add the whole numbers separately
  2. Add the fractions (convert to like fractions if needed)
  3. If the fraction sum is improper, convert it to a mixed number and add to the whole number
  4. Simplify if possible

Example 1:

\[ 2\frac{1}{3} + 3\frac{1}{3} \]

Add whole numbers: 2 + 3 = 5

Add fractions: \(\frac{1}{3} + \frac{1}{3} = \frac{2}{3}\)

Result: \(5\frac{2}{3}\)

Example 2:

\[ 1\frac{1}{2} + 2\frac{2}{3} \]

Convert to improper fractions:

\[ \frac{3}{2} + \frac{8}{3} \]

LCD is 6:

\[ \frac{9}{6} + \frac{16}{6} = \frac{25}{6} = 4\frac{1}{6} \]

Subtracting Mixed Numbers

To subtract mixed numbers:

  1. Subtract the whole numbers separately
  2. Subtract the fractions (convert to like fractions if needed)
  3. If the first fraction is smaller than the second, borrow 1 from the whole number
  4. Simplify if possible

Example 1:

\[ 3\frac{3}{4} - 1\frac{1}{4} \]

Subtract whole numbers: 3 - 1 = 2

Subtract fractions: \(\frac{3}{4} - \frac{1}{4} = \frac{2}{4} = \frac{1}{2}\)

Result: \(2\frac{1}{2}\)

Example 2 (with borrowing):

\[ 5\frac{1}{4} - 2\frac{3}{4} \]

Borrow 1 from 5 (convert to \(\frac{4}{4}\)):

\[ 4\frac{5}{4} - 2\frac{3}{4} \]

Subtract: \(4 - 2 = 2\) and \(\frac{5}{4} - \frac{3}{4} = \frac{2}{4} = \frac{1}{2}\)

Result: \(2\frac{1}{2}\)

Practice Exercise

Question 1

Add: \(\frac{2}{5} + \frac{3}{5}\)

\[ \frac{2 + 3}{5} = \frac{5}{5} = 1 \]

Question 2

Subtract: \(\frac{7}{8} - \frac{3}{8}\)

\[ \frac{7 - 3}{8} = \frac{4}{8} = \frac{1}{2} \]

Question 3

Add: \(\frac{1}{3} + \frac{1}{4}\)

LCD is 12:

\[ \frac{4}{12} + \frac{3}{12} = \frac{7}{12} \]

Question 4

Add: \(2\frac{1}{2} + 3\frac{1}{4}\)

Convert to improper fractions:

\[ \frac{5}{2} + \frac{13}{4} \]

LCD is 4:

\[ \frac{10}{4} + \frac{13}{4} = \frac{23}{4} = 5\frac{3}{4} \]

Question 5

Subtract: \(4\frac{1}{3} - 2\frac{2}{3}\)

Borrow 1 from 4:

\[ 3\frac{4}{3} - 2\frac{2}{3} = 1\frac{2}{3} \]

Question 6

Subtract: \(\frac{5}{6} - \frac{1}{2}\)

LCD is 6:

\[ \frac{5}{6} - \frac{3}{6} = \frac{2}{6} = \frac{1}{3} \]
