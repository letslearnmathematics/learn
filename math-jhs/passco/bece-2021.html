Question 3


  1. If \(\mathbf{r} = \begin{pmatrix} -4 \\ -5 \end{pmatrix}\) and \(\mathbf{m} = \begin{pmatrix} -1 \\ -2 \end{pmatrix}\), find \(\mathbf{p}\) given that \(\mathbf{p} = \mathbf{r} - \mathbf{m}\).

    2. Solution

    $\mathbf{r} = \begin{pmatrix} -4 \\ -5 \end{pmatrix}$

    $\mathbf{m} = \begin{pmatrix} -1 \\ -2 \end{pmatrix}$

    $\mathbf{p} = \mathbf{r} - \mathbf{m}$

    $\Rightarrow \mathbf{p} = \begin{pmatrix} -4 \\ -5 \end{pmatrix} - \begin{pmatrix} -1 \\ -2 \end{pmatrix} $

    $\Rightarrow \mathbf{p} = \begin{pmatrix} -4 - (-1) \\ -5 - (-2) \end{pmatrix} $

    $\Rightarrow \mathbf{p} = \begin{pmatrix} -4 + 1 \\ -5 + 2 \end{pmatrix} $

    $\Rightarrow \mathbf{p} = \begin{pmatrix} -3 \\ -3 \end{pmatrix} $



  2. The sum of two numbers is 81. If the second number is twice the first, find the second number.

    3. Solution

    let the first number $ = x $

    Hence, the second number $ = 2x$

    $\Rightarrow x + 2x = 81$

    $\Rightarrow 3x = 81$

    $\Rightarrow \dfrac{3x}{3} = \dfrac{81}{3}$

    $\Rightarrow x = 27$

    Second number $\Rightarrow 2x$

    Second number $\Rightarrow 2(27)$

    Second number $\Rightarrow 54$

    $\underline{\therefore \text{the second number is 54.}}$

    width $= 4$ m
