Introduction

Rational numbers are numbers that can be expressed as a fraction \(\frac{a}{b}\) where \(a\) and \(b\) are integers and \(b \neq 0\). They include integers, fractions, and terminating or repeating decimals.

In this lesson, we will explore the relationships between different types of rational numbers using set theory concepts of union and intersection. We'll also learn how to represent factors of numbers using Venn diagrams.

Union (\( A \cup B \)):

Combines all elements from both sets

Intersection (\( A \cap B \)):

Contains only elements common to both sets

Real Number System

Hierarchy of Real Numbers

The real number system can be represented using nested sets to show how different number types relate:

Key relationships:

  • Natural numbers (\( \mathbb{N} \)) ⊂ Whole numbers (\( \mathbb{W} \)) ⊂ Integers (\( \mathbb{Z} \)) ⊂ Rational numbers (\( \mathbb{Q} \)) ⊂ Real numbers (\( \mathbb{R} \))
  • Irrational numbers (\( \mathbb{I} \)) are separate but also part of real numbers

Set Operations with Number Types

We can use union and intersection to explore relationships between number sets:

Example 1: Relationship between integers and whole numbers

\[ \mathbb{W} \cap \mathbb{Z} = \mathbb{W} \]

This shows whole numbers are a subset of integers

Example 2: Relationship between rational and irrational numbers

\[ \mathbb{Q} \cap \mathbb{I} = \emptyset \]

This shows they have no common elements

Factors on Venn Diagrams

Representing Factors

Venn diagrams can visually represent common factors between numbers:

Steps to Create Factor Venn Diagrams

  1. List all factors of each number
  2. Identify common factors (intersection)
  3. Place unique factors in separate circles
  4. Place common factors in overlapping area

Example: Factors of 12 and 18

Factors of 12: {1, 2, 3, 4, 6, 12}

Factors of 18: {1, 2, 3, 6, 9, 18}

Common factors: {1, 2, 3, 6}

Finding HCF Using Venn Diagrams

The highest common factor (HCF) is the largest number in the intersection:

From our example of 12 and 18:

\[ \text{HCF} = 6 \]

This is the greatest number in the overlapping section

Problem Solving

Applying Union and Intersection

Let's solve problems combining number relationships and set operations:

Problem 1:

Given:

\( A = \{x : x \text{ is a factor of 24}\} \)

\( B = \{x : x \text{ is a factor of 36}\} \)

Find \( A \cup B \) and \( A \cap B \)

Solution:

Factors of 24: {1, 2, 3, 4, 6, 8, 12, 24}

Factors of 36: {1, 2, 3, 4, 6, 9, 12, 18, 36}

\( A \cup B = \{1, 2, 3, 4, 6, 8, 9, 12, 18, 24, 36\} \)

\( A \cap B = \{1, 2, 3, 4, 6, 12\} \)

Problem 2:

Represent the relationship between:

Natural numbers (\( \mathbb{N} \)), Whole numbers (\( \mathbb{W} \)), Integers (\( \mathbb{Z} \)), and Rational numbers (\( \mathbb{Q} \))

Solution:

This is a nested relationship:

\( \mathbb{N} \subset \mathbb{W} \subset \mathbb{Z} \subset \mathbb{Q} \)
