Introduction

Significant figures (or significant digits) refer to the digits that give meaning to a particular number or contributes to the number’s precision. These include all non-zero digits, any zeros between significant digits, and trailing zeros in the decimal part.

In this lesson, we will learn how to:

  1. Express integers to a specified number of significant figures
  2. Express decimal numbers to a given number of decimal places

Expressing Numbers to Significant Figures

To express a number to a given number of significant figures:

  1. Count the number of significant figures needed
  2. Identify the digit at the position where you'll round
  3. Look at the next digit (the first non-significant digit)
  4. If it's 5 or more, round up the last significant digit
  5. If it's less than 5, keep the last significant digit unchanged
  6. Replace any remaining digits with zeros if necessary (for numbers without decimal points)

Rules for Decimal Places

  1. Count the digits after the decimal point
  2. Identify the digit at the specified decimal place
  3. Look at the next digit (the first non-included digit)
  4. If it's 5 or more, round up the last decimal digit
  5. If it's less than 5, keep the last decimal digit unchanged
  6. Drop any remaining digits after the specified decimal places

Relationship Between Significant Figures and Decimal Places

While both concepts deal with precision, they serve different purposes:

  • Significant figures count all meaningful digits in a number, regardless of decimal point position
  • Decimal places count only digits after the decimal point
