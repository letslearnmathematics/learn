Introduction to Vectors

A vector is a quantity that has both magnitude (size) and direction. Unlike scalars which only have magnitude, vectors tell us not just how much but also which way.

In the Cartesian plane (the x-y coordinate system), we can represent points as position vectors. This means we can describe the exact location of any point using both its horizontal (x) and vertical (y) components.

For example, the point P(3,4) can be represented as a vector:

\[ \vec{OP} = \begin{pmatrix} 3 \\ 4 \end{pmatrix} \]

where O is the origin (0,0).

Key Characteristics:

  • Vectors have both magnitude and direction
  • Position vectors start from the origin (0,0)
  • Vectors can be represented in component form (x,y)

Position Vectors

Understanding Position Vectors

A position vector describes the location of a point relative to the origin (0,0) in the Cartesian plane. It's represented by an arrow starting at the origin and ending at the point.

The components of the vector correspond to the x and y coordinates of the point.

Representing Points as Vectors

Any point P(a,b) in the plane can be written as a position vector:

\[ \vec{OP} = \begin{pmatrix} a \\ b \end{pmatrix} \]

This means:

  • Move 'a' units along the x-axis
  • Move 'b' units along the y-axis

Drawing Position Vectors

To draw a position vector for point P(a,b):

  1. Start at the origin O(0,0)
  2. Draw an arrow to point P(a,b)
  3. Label the vector as \(\vec{OP}\)

Vector Notation

Different Ways to Represent Vectors

Vectors can be represented in several equivalent ways. The most common notations are:

Column Vector Form:

\[ \vec{v} = \begin{pmatrix} x \\ y \end{pmatrix} \]

Unit Vector Form:

\[ \vec{v} = x\mathbf{i} + y\mathbf{j} \]

where \(\mathbf{i}\) and \(\mathbf{j}\) are unit vectors in the x and y directions

Ordered Pair Form:

\[ \vec{v} = (x, y) \]

Converting Between Notations

All these forms represent the same vector and can be converted between each other:

Example: Convert \(\begin{pmatrix} 3 \\ -1 \end{pmatrix}\) to unit vector form

\[ \begin{pmatrix} 3 \\ -1 \end{pmatrix} = 3\mathbf{i} - \mathbf{j} \]

Magnitude of Vectors

Calculating Magnitude

The magnitude (length) of a vector \(\vec{v} = \begin{pmatrix} x \\ y \end{pmatrix}\) is calculated using the Pythagorean theorem:

\[ |\vec{v}| = \sqrt{x^2 + y^2} \]

Example: Find the magnitude of \(\vec{v} = \begin{pmatrix} 3 \\ 4 \end{pmatrix}\)

\[ |\vec{v}| = \sqrt{3^2 + 4^2} = \sqrt{9 + 16} = \sqrt{25} = 5 \]

Direction of Vectors

The direction of a vector can be described by the angle it makes with the positive x-axis, calculated as:

\[ \theta = \tan^{-1}\left(\frac{y}{x}\right) \]

Example: Find the direction of \(\vec{v} = \begin{pmatrix} 1 \\ 1 \end{pmatrix}\)

\[ \theta = \tan^{-1}\left(\frac{1}{1}\right) = 45^\circ \]

Practice Exercise

Question 1

Find the magnitude of the vector \(\begin{pmatrix} 6 \\ 8 \end{pmatrix}\).

\[ |\vec{v}| = \sqrt{6^2 + 8^2} = \sqrt{36 + 64} = \sqrt{100} = 10 \]

Question 2

Convert the vector \(2\mathbf{i} - 5\mathbf{j}\) to column vector form.

\[ 2\mathbf{i} - 5\mathbf{j} = \begin{pmatrix} 2 \\ -5 \end{pmatrix} \]

Question 3

Find the direction (angle) of the vector \(\begin{pmatrix} \sqrt{3} \\ 1 \end{pmatrix}\).

\[ \theta = \tan^{-1}\left(\frac{1}{\sqrt{3}}\right) = 30^\circ \]

Real-world Applications

Where Vectors Are Used

Position vectors have many practical applications in science and engineering:

  • Physics: Describing forces, velocity, and acceleration
  • Navigation: GPS systems use vectors to determine position
  • Computer Graphics: Positioning objects in 2D and 3D space
  • Robotics: Controlling movement and positioning of robotic arms

Navigation Example

If a ship is at position (3,4) relative to port (0,0), its position vector is:

\[ \vec{v} = \begin{pmatrix} 3 \\ 4 \end{pmatrix} \]

The magnitude \(|\vec{v}| = 5\) km tells how far the ship is from port, and the direction \(\theta \approx 53.13^\circ\) tells its bearing from the port.
