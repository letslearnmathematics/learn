Introduction

Linear relations can be represented graphically as straight lines on a coordinate plane. When we have two linear relations, their graphs can help us find missing elements in ordered pairs.

In this lesson, we'll learn how to use the graphs of two linear relations to determine missing x or y values in ordered pairs. This skill is essential for solving systems of equations and understanding how variables relate to each other.

Given two linear relations:

\[ y = 2x + 1 \] \[ y = -x + 4 \]

We can graph both lines and find their point of intersection to determine the (x, y) pair that satisfies both equations.

Finding Missing Elements of Ordered Pairs

Method for Finding Missing Elements

To find missing elements in ordered pairs using graphs of two linear relations:

  1. Graph both linear equations on the same coordinate plane
  2. Identify the point(s) where the graphs intersect (if they do)
  3. For a given x-value, find the corresponding y-values on both graphs
  4. For a given y-value, find the corresponding x-values on both graphs

Example 1: Find missing y-value

Given the graphs of y = 2x + 1 and y = -x + 4, find y when x = 1:

  • For y = 2x + 1: y = 2(1) + 1 = 3
  • For y = -x + 4: y = -(1) + 4 = 3

Both relations give y = 3 when x = 1, so the ordered pair is (1, 3).

Example 2: Find missing x-value

Given the same graphs, find x when y = 2:

  • For y = 2x + 1: 2 = 2x + 1 ⇒ x = 0.5
  • For y = -x + 4: 2 = -x + 4 ⇒ x = 2

The ordered pairs are (0.5, 2) and (2, 2) for each relation respectively.

Using Intersection Points

Finding the Solution to Two Linear Relations

The point where two linear graphs intersect represents the (x, y) pair that satisfies both equations simultaneously. This is the solution to the system of equations.

Given the graphs of:

\[ y = 2x + 1 \] \[ y = -x + 4 \]

The intersection point is (1, 3), which means:

\[ 3 = 2(1) + 1 \] \[ 3 = -(1) + 4 \]

Both equations are satisfied when x = 1 and y = 3.

Graphical Interpretation

When working with graphs of two linear relations:

  • One intersection point: The equations have one unique solution (the lines intersect)
  • No intersection point: The equations have no solution (the lines are parallel)
  • Infinite intersection points: The equations have infinitely many solutions (the lines coincide)

Parallel Lines Example

y = 2x + 1 and y = 2x + 3 are parallel (same slope) and never intersect.

Coinciding Lines Example

y = 2x + 1 and 2y = 4x + 2 represent the same line (infinite solutions).

Practice Exercise

Question 1

Given the graphs of y = x + 2 and y = -2x + 5, find the missing y-value when x = 1.

For y = x + 2: y = 1 + 2 = 3

For y = -2x + 5: y = -2(1) + 5 = 3

The missing y-value is 3 for both relations when x = 1.

Question 2

Using the same graphs (y = x + 2 and y = -2x + 5), find the x-value(s) when y = 4.

For y = x + 2: 4 = x + 2 ⇒ x = 2

For y = -2x + 5: 4 = -2x + 5 ⇒ x = 0.5

The x-values are 2 and 0.5 for the respective relations when y = 4.

Question 3

Find the point of intersection for y = 3x - 1 and y = x + 3.

Set the equations equal to each other: 3x - 1 = x + 3

Solve for x: 2x = 4 ⇒ x = 2

Find y: y = 3(2) - 1 = 5

The intersection point is (2, 5).

Question 4

Given the graphs of y = ½x + 3 and y = -x + 6, find the missing y-value when x = 4.

For y = ½x + 3: y = ½(4) + 3 = 5

For y = -x + 6: y = -(4) + 6 = 2

The y-values are 5 and 2 for the respective relations when x = 4.

Question 5

Determine if the lines y = 4x - 2 and y = 4x + 5 intersect. If not, explain why.

Both lines have the same slope (4) but different y-intercepts (-2 and 5).

This means the lines are parallel and never intersect.

There is no solution to this system of equations.
