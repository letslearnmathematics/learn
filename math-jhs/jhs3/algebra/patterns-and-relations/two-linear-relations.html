Introduction

A linear relation is a relationship between two variables that forms a straight line when graphed. These relations can be represented in various ways including equations, tables of values, and graphs.

In this lesson, we will learn how to:

  1. Construct tables of values for two linear relations
  2. Draw graphs of these relations
  3. Interpret the graphs to understand the relationship between variables

Example of a linear relation:

\[ y = 2x + 1 \]

This equation represents a straight line with slope 2 and y-intercept 1.

Table of Values

Constructing Tables of Values

A table of values shows corresponding x and y values for a given relation. To construct one:

  1. Choose several x-values (usually integers)
  2. Substitute each x-value into the equation to find the corresponding y-value
  3. Organize the pairs in a table

Example 1

Construct a table of values for the relation \( y = 2x + 1 \) for \( x \) values from -2 to 2.

x y = 2x + 1
-2 2(-2) + 1 = -3
-1 2(-1) + 1 = -1
0 2(0) + 1 = 1
1 2(1) + 1 = 3
2 2(2) + 1 = 5

Example 2

Construct a table of values for the relation \( y = -x + 3 \) for \( x \) values from -1 to 3.

x y = -x + 3
-1 -(-1) + 3 = 4
0 -(0) + 3 = 3
1 -(1) + 3 = 2
2 -(2) + 3 = 1
3 -(3) + 3 = 0

Practice Exercise

Question 1

Construct a table of values for \( y = 3x - 2 \) for \( x \) values from -2 to 2.

x y = 3x - 2
-2 3(-2) - 2 = -8
-1 3(-1) - 2 = -5
0 3(0) - 2 = -2
1 3(1) - 2 = 1
2 3(2) - 2 = 4

Question 2

Construct a table of values for \( y = \frac{1}{2}x + 1 \) for \( x \) values from -4 to 4 in steps of 2.

x y = ½x + 1
-4 ½(-4) + 1 = -1
-2 ½(-2) + 1 = 0
0 ½(0) + 1 = 1
2 ½(2) + 1 = 2
4 ½(4) + 1 = 3

Graphing Relations

Drawing Graphs

To graph a linear relation from its table of values:

  1. Draw and label your x and y axes
  2. Choose an appropriate scale for both axes
  3. Plot each (x, y) pair from your table as a point
  4. Connect the points with a straight line
  5. Label your graph with the equation of the line

Example 1

Graph the relation \( y = 2x + 1 \) using the table of values we created earlier.

From our table:

x y
-2 -3
-1 -1
0 1
1 3
2 5

The graph would show these points connected by a straight line.

Example 2

Graph the relation \( y = -x + 3 \) using its table of values.

From our table:

x y
-1 4
0 3
1 2
2 1
3 0

The graph would show these points connected by a straight line.
