Introduction

Linear inequalities are mathematical expressions that show the relationship between two quantities where one is not necessarily equal to the other. Unlike equations which use an equals sign (=), inequalities use inequality signs to show these relationships.

In this lesson, we will learn how to represent solution sets of linear inequalities on the number line and cartesian plane. Understanding inequalities is crucial for solving real-world problems where exact equality is not always possible or necessary.

Inequality Signs

Understanding Inequality Symbols

There are four main inequality signs used in mathematics:

Less Than (<)

Indicates the value on the left is smaller than the value on the right.

\[ x < 5 \]

Means "x is less than 5"

Greater Than (>)

Indicates the value on the left is larger than the value on the right.

\[ x > 3 \]

Means "x is greater than 3"

Less Than or Equal To (≤)

Indicates the value on the left is smaller than or equal to the value on the right.

\[ x \leq 4 \]

Means "x is less than or equal to 4"

Greater Than or Equal To (≥)

Indicates the value on the left is larger than or equal to the value on the right.

\[ x \geq 2 \]

Means "x is greater than or equal to 2"

Key Points to Remember

  • The open circle (○) is used for < and > inequalities (not including the endpoint)
  • The closed circle (•) is used for ≤ and ≥ inequalities (including the endpoint)
  • The arrow always points in the direction of the smaller values for less than, and larger values for greater than
  • When multiplying or dividing both sides by a negative number, the inequality sign reverses

Graphing Linear Inequalities in One Variable on a Number Line

Steps for Graphing on Number Line

To graph a linear inequality on a number line:

  1. Solve the inequality for the variable (if necessary)
  2. Identify the critical value (the number in the inequality)
  3. Determine whether to use an open or closed circle based on the inequality sign
  4. Draw the circle at the critical value on the number line
  5. Draw an arrow extending in the appropriate direction

Example 1: Graphing x > 2

1. Critical value is 2

2. Use open circle (>)

3. Arrow points to the right (greater than)

Example 2: Graphing x ≤ -1

1. Critical value is -1

2. Use closed circle (≤)

3. Arrow points to the left (less than)

Solve and Graph Linear Inequalities on a Number Line

Solving Linear Inequalities

Solving linear inequalities follows similar steps to solving equations, with one important exception:

  1. Simplify both sides of the inequality if needed (distribute, combine like terms)
  2. Use inverse operations to isolate the variable
  3. Remember: When multiplying or dividing both sides by a negative number, reverse the inequality sign
  4. Graph the solution on the number line

Example 1: Solve and graph 2x + 3 < 7

Solve and Graph Linear Inequalities on a Cartesian Plane

Graphing Two-Variable Inequalities

To graph linear inequalities in two variables (like y > 2x + 1) on the Cartesian plane:

  1. Graph the corresponding equation (e.g., y = 2x + 1) as a boundary line
  2. Use a dashed line for < or > (not including the line)
  3. Use a solid line for ≤ or ≥ (including the line)
  4. Choose a test point not on the line (usually (0,0) is easiest)
  5. Shade the region that satisfies the inequality

Example 1: Graph y > 2x + 1

1. Graph y = 2x + 1 with a dashed line

2. Test point (0,0): 0 > 2(0)+1 → 0 > 1 (False)

3. Shade the region above the line

Example 2: Graph 3x - 2y ≤ 6

1. Solve for y: y ≥ (3/2)x - 3

2. Graph y = (3/2)x - 3 with a solid line

3. Test point (0,0): 0 ≥ -3 (True)

4. Shade the region above the line

Practice Exercise

Question 1

Graph the inequality y < -x + 4 on the Cartesian plane.

1. Graph y = -x + 4 with a dashed line

2. Test point (0,0): 0 < 4 (True)

3. Shade the region below the line

Question 2

Graph the inequality 2x + y ≥ 5 on the Cartesian plane.

1. Solve for y: y ≥ -2x + 5

2. Graph y = -2x + 5 with a solid line

3. Test point (0,0): 0 ≥ 5 (False)

4. Shade the region above the line

Question 3

1. Solve for y: y < (1/3)x - 2

2. Graph y = (1/3)x - 2 with a dashed line

3. Test point (0,0): 0 < -2 (False)

4. Shade the region below the line
