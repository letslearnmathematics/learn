Introduction

Data organization is the process of arranging collected data in a meaningful way to make it easier to understand, analyze, and interpret. Proper data organization helps us identify patterns, trends, and relationships in the data.

Data can be either grouped (organized into categories or intervals) or ungrouped (raw data without any organization). We will learn various methods to present data visually and analyze it to solve problems.

Frequency Tables

Creating Frequency Tables

A frequency table shows how often each value or category appears in a dataset. It consists of:

  • Data values or class intervals
  • Tally marks for counting
  • Frequency count

Steps to Create

  1. List all possible values or create class intervals
  2. Use tally marks to count occurrences
  3. Sum the tally marks for each category
  4. Present in a clear table format

Example

Test scores of 20 students:

\[ 65, 72, 68, 85, 90, 72, 65, 78, 85, 65, 72, 90, 85, 78, 65, 72, 85, 90, 78, 85 \]

Score Tally Frequency
65 |||| 4
68 | 1
72 |||| 4
78 ||| 3
85 |||| | 6
90 || 2

Bar Graphs

Understanding Bar Graphs

Bar graphs use rectangular bars to represent categorical data. The length of each bar corresponds to the value it represents.

Components

  • Title: Describes what the graph represents
  • X-axis: Shows categories being compared
  • Y-axis: Shows the scale of measurement
  • Bars: Represent the quantities
  • Legend: Explains colors/patterns if needed

Pie Charts

Understanding Pie Charts

Pie charts show proportions of a whole. Each category is represented by a slice whose size corresponds to its proportion.

Creating Pie Charts

  1. Calculate the total of all values
  2. For each category, calculate its percentage of the total
  3. Multiply each percentage by 360° to get the angle
  4. Draw a circle and divide it according to the angles
  5. Label each sector with its category and percentage

Example

Budget allocation for a school:

  • Teachers' salaries: \$50,000
  • Facilities: \$30,000
  • Supplies: \$15,000
  • Other: \$5,000

Total = \$100,000

Practice Exercise

Question 1

A survey of 120 students showed their preferred sports: Football (45), Basketball (30), Volleyball (25), and Athletics (20). Create a pie chart to represent this data.

Calculations:

  • Football: (45/120) × 360° = 135°
  • Basketball: (30/120) × 360° = 90°
  • Volleyball: (25/120) × 360° = 75°
  • Athletics: (20/120) × 360° = 60°

Line Graphs

Understanding Line Graphs

Line graphs show how data changes over time. Points are plotted and connected with straight lines to show trends.

When to Use

  • Tracking changes over short and long periods
  • Comparing changes over same period for multiple groups
  • Showing trends clearly

Advanced Representations

Specialized Data Visualizations

Beyond basic graphs, there are more specialized ways to represent data that can provide deeper insights.

Infographics

Combine visuals and text to present complex information quickly and clearly.

Waffle Diagrams

Show parts of a whole using a grid of small squares where each square represents a percentage.

Box and Whisker Plots

Display data distribution through quartiles, highlighting median, range, and outliers.

Stem and Leaf Plots

Show quantitative data while preserving individual data points.
