Introduction

Extreme values (also called outliers) are data points that are significantly higher or lower than most of the other values in a data set. These values can have a noticeable effect on measures of central tendency - mean, median, and mode.

Consider this data set of test scores:

\[ 78, 82, 85, 88, 90, 92, 95 \]

Now let's add an extreme low score:

\[ 25, 78, 82, 85, 88, 90, 92, 95 \]

And an extreme high score:

\[ 78, 82, 85, 88, 90, 92, 95, 150 \]

Key Terms:

  • Mean: The average of all values
  • Median: The middle value when data is ordered
  • Mode: The most frequent value
  • Outlier: An extreme value that differs significantly from others

Effect on Mean

How Extreme Values Affect the Mean

The mean is the measure most affected by extreme values because it takes into account every value in the data set when calculating the average.

Original data set: 78, 82, 85, 88, 90, 92, 95

\[ \text{Mean} = \frac{78 + 82 + 85 + 88 + 90 + 92 + 95}{7} = \frac{610}{7} \approx 87.14 \]

With extreme low (25): 25, 78, 82, 85, 88, 90, 92, 95

\[ \text{Mean} = \frac{25 + 78 + 82 + 85 + 88 + 90 + 92 + 95}{8} = \frac{635}{8} \approx 79.38 \]

The mean decreased by 7.76 points

With extreme high (150): 78, 82, 85, 88, 90, 92, 95, 150

\[ \text{Mean} = \frac{78 + 82 + 85 + 88 + 90 + 92 + 95 + 150}{8} = \frac{760}{8} = 95 \]

The mean increased by 7.86 points

Key Observation:

The mean is pulled in the direction of the extreme value:

  • Low extreme pulls mean down
  • High extreme pulls mean up

Effect on Median

How Extreme Values Affect the Median

The median is less affected by extreme values because it only depends on the middle value(s) in an ordered data set, not all values.

Original data set (7 values): 78, 82, 85, 88, 90, 92, 95

Median = 88 (4th value)

With extreme low (25) - now 8 values: 25, 78, 82, 85, 88, 90, 92, 95

Median = (85 + 88)/2 = 86.5

Median decreased by 1.5 points

With extreme high (150) - now 8 values: 78, 82, 85, 88, 90, 92, 95, 150

Median = (88 + 90)/2 = 89

Median increased by 1 point

Key Observation:

The median is more resistant to extreme values than the mean:

  • Changes much less than the mean when extremes are added
  • Only changes based on position in ordered list, not value magnitude

Effect on Mode

How Extreme Values Affect the Mode

The mode is generally not affected by extreme values unless the extreme value happens to be the most frequent value in the data set.

Original data set: 78, 82, 85, 88, 90, 92, 95

Mode = No mode (all values unique)

Modified data set with repeated value: 78, 82, 85, 85, 88, 90, 92, 95

Mode = 85

Adding extreme low (25): 25, 78, 82, 85, 85, 88, 90, 92, 95

Mode remains = 85

Adding extreme high (150): 78, 82, 85, 85, 88, 90, 92, 95, 150

Mode remains = 85

Only changes if extreme becomes most frequent: 78, 82, 85, 85, 88, 90, 92, 95, 95, 95

New mode = 95

Key Observation:

The mode is least affected by extreme values:

  • Only changes if extreme value becomes the new most frequent value
  • Unaffected by magnitude of extreme values

Comparing the Effects

Summary of Effects

Measure Effect of Extreme Low Effect of Extreme High Resistance to Extremes
Mean Decreases significantly Increases significantly Least resistant
Median Decreases slightly Increases slightly Moderately resistant
Mode Usually no effect Usually no effect Most resistant

When to Use Each Measure:

  • Mean: Best for data without extreme values
  • Median: Best when extreme values are present
  • Mode: Best for categorical data or identifying most common value

Practice Exercise

Question 1

Consider the data set: 12, 15, 18, 20, 22, 24, 26

a) Calculate the mean, median, and mode

b) Add an extreme value of 100 and recalculate

c) Compare the changes

Original:

Mean = (12+15+18+20+22+24+26)/7 = 137/7 ≈ 19.57

Median = 20 (4th value)

Mode = No mode

With extreme (100):

Mean = (12+15+18+20+22+24+26+100)/8 = 237/8 = 29.625

Median = (20+22)/2 = 21

Mode = No mode

Changes:

Mean increased by ≈10.06

Median increased by 1

Mode unchanged

Question 2

The monthly salaries (in Gh₵) of employees at a small company are: 1200, 1250, 1300, 1350, 1400, 1450, 1500, 1600, 1800, 5000

a) Identify the extreme value

b) Calculate mean and median with and without the extreme value

c) Which measure better represents typical salaries?

Extreme value: 5000 (much higher than others)

With all values:

Mean = (sum)/10 = 17850/10 = 1785

Median = (1400+1450)/2 = 1425

Without extreme (5000):

Mean = (12850)/9 ≈ 1427.78

Median = 1400 (5th value)

Better measure: Median (less affected by extreme)

Question 3

A teacher records the following test scores: 55, 60, 65, 70, 75, 80, 85, 90, 95, 30

a) Which measure changes most when the 30 is removed?

b) Explain why this happens

With all values:

Mean = 705/10 = 70.5

Median = (75+80)/2 = 77.5

Mode = No mode

Without 30:

Mean = 675/9 = 75

Median = 75 (5th value)

Mode = No mode

Changes:

Mean increased by 4.5

Median decreased by 2.5

Mode unchanged

Explanation: The mean changes most because it's calculated using all values, while the median only depends on the middle values.
