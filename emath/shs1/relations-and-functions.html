Relations

A relation is a set of ordered pairs that establishes a connection between elements of two sets. In mathematics, we often write relations as \( (x, y) \) where \( x \) is related to \( y \).

Example of a relation:

\[ R = \{(1, 2), (2, 4), (3, 6), (4, 8)\} \]

This can be described by the rule \( y = 2x \)

Types of Relations:

  • One-to-One: Each x relates to only one y, and each y relates to only one x
  • One-to-Many: An x relates to multiple y values
  • Many-to-One: Multiple x values relate to the same y
  • Many-to-Many: Multiple x relate to multiple y values

Representation:

Relations can be represented as:

  • Ordered pairs
  • Arrow diagrams
  • Graphs
  • Equations

Functions

A function is a special type of relation where each input (x-value) has exactly one output (y-value). Functions are often written as \( f(x) = \) expression.

Example of a function:

\[ f(x) = 2x + 3 \]

For \( x = 1 \), \( f(1) = 2(1) + 3 = 5 \)

Vertical Line Test:

A graph represents a function if any vertical line intersects the graph at most once.

Function Notation:

  • \( f: x \mapsto 2x + 3 \) (Mapping notation)
  • \( f(x) = 2x + 3 \) (Function notation)
  • \( y = 2x + 3 \) (Equation form)

Identifying Functions

Determine whether these relations are functions:

1. \( \{(1, 2), (2, 4), (3, 6)\} \)

This is a function because each x-value has exactly one y-value.

2. \( \{(1, 2), (1, 4), (2, 5)\} \)

This is not a function because x=1 has two different y-values (2 and 4).

Largest Possible Domain of a Function

The domain of a function is the set of all possible input values (x-values) for which the function is defined.

For \( f(x) = \frac{1}{x} \), the domain is all real numbers except 0:

\[ \text{Domain} = \{x \in \mathbb{R} \mid x \neq 0\} \]

Common Restrictions:

  • Denominators cannot be zero
  • Expressions under square roots must be ≥ 0
  • Logarithms must have positive arguments

Domain Examples

Find the largest possible domain for:

\[ f(x) = \sqrt{x - 2} \]

The expression under the square root must be ≥ 0:

\[ x - 2 \geq 0 \] \[ x \geq 2 \]

Domain: \( \{x \in \mathbb{R} \mid x \geq 2\} \) or \( [2, \infty) \)

Find the largest possible domain for:

\[ g(x) = \frac{1}{x^2 - 4} \]

The denominator cannot be zero:

\[ x^2 - 4 \neq 0 \] \[ x \neq \pm 2 \]

Domain: \( \{x \in \mathbb{R} \mid x \neq -2, x \neq 2\} \)

Range of a Function

The range of a function is the set of all possible output values (y-values) that result from using the domain.

For \( f(x) = x^2 \), the range is all real numbers ≥ 0:

\[ \text{Range} = \{y \in \mathbb{R} \mid y \geq 0\} \]

Finding Range:

  • Analyze the function's behavior (graphically or algebraically)
  • For quadratic functions, find the vertex
  • For rational functions, find horizontal asymptotes

Range Examples

Find the range of:

\[ f(x) = 2x + 3 \]

This is a linear function with no restrictions:

Range: \( \mathbb{R} \) (all real numbers)

Find the range of:

\[ g(x) = x^2 - 4x + 5 \]

Complete the square:

\[ g(x) = (x^2 - 4x + 4) + 1 = (x - 2)^2 + 1 \]

The minimum value is 1 (when x=2):

Range: \( \{y \in \mathbb{R} \mid y \geq 1\} \) or \( [1, \infty) \)

Inverse of a Function

The inverse function \( f^{-1} \) reverses the original function \( f \), swapping x and y values. For \( f^{-1} \) to exist, \( f \) must be one-to-one.

Given \( f(x) = 2x + 3 \), find \( f^{-1}(x) \):

  1. Let \( y = 2x + 3 \)
  2. Swap x and y: \( x = 2y + 3 \)
  3. Solve for y: \( y = \frac{x - 3}{2} \)
\[ f^{-1}(x) = \frac{x - 3}{2} \]

Horizontal Line Test:

A function has an inverse if any horizontal line intersects its graph at most once.

Domain and Range Swap:

For \( f \) with domain D and range R, \( f^{-1} \) has domain R and range D.

Non-Invertible Functions

Functions that are not one-to-one (like quadratic functions) can be made invertible by restricting their domain.

For \( f(x) = x^2 \), if we restrict the domain to \( x \geq 0 \):

\[ f^{-1}(x) = \sqrt{x} \]

Composite Functions

A composite function combines two functions where the output of one becomes the input of the other, written as \( (f \circ g)(x) = f(g(x)) \).

Given \( f(x) = x + 2 \) and \( g(x) = 3x \), find \( f \circ g \) and \( g \circ f \):

\[ (f \circ g)(x) = f(g(x)) = f(3x) = 3x + 2 \] \[ (g \circ f)(x) = g(f(x)) = g(x + 2) = 3(x + 2) = 3x + 6 \]

Note: \( f \circ g \neq g \circ f \) in most cases

Domain Consideration:

The domain of \( f \circ g \) is all x in the domain of g where g(x) is in the domain of f.

Composite Function Examples

Given \( f(x) = \sqrt{x} \) and \( g(x) = x - 4 \), find \( f \circ g \) and its domain.

First find the composition:

\[ (f \circ g)(x) = f(g(x)) = \sqrt{x - 4} \]

Now determine the domain:

For \( \sqrt{x - 4} \), we need \( x - 4 \geq 0 \), so \( x \geq 4 \)

Domain: \( [4, \infty) \)

If \( h(x) = (x^2 + 1)^3 \), express h as a composition of two simpler functions.

Let \( f(x) = x^3 \) and \( g(x) = x^2 + 1 \)

Then \( h(x) = (f \circ g)(x) = f(g(x)) = f(x^2 + 1) = (x^2 + 1)^3 \)

Practice Exercise

Question 1

Determine whether \( \{(1, 5), (2, 5), (3, 6)\} \) represents a function. If yes, find its range.

This is a function (each x has exactly one y). Range: \( \{5, 6\} \)

Question 2

Find the largest possible domain for \( f(x) = \frac{1}{\sqrt{x - 1}} \).

We need \( x - 1 > 0 \) (denominator can't be zero and square root must be positive):

\[ x > 1 \]

Domain: \( (1, \infty) \)

Question 3

Given \( f(x) = 2x - 1 \) and \( g(x) = x^2 + 1 \), find \( (f \circ g)(2) \).

First find \( g(2) \):

\[ g(2) = 2^2 + 1 = 5 \]

Then find \( f(g(2)) = f(5) \):

\[ f(5) = 2(5) - 1 = 9 \]

So \( (f \circ g)(2) = 9 \)
