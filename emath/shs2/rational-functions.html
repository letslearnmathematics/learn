Definition of Rational Functions

A rational function is any function that can be expressed as the ratio of two polynomial functions:

\[ f(x) = \frac{P(x)}{Q(x)} \]

where \( P(x) \) and \( Q(x) \) are polynomials and \( Q(x) \neq 0 \).

Examples:

  • \( f(x) = \frac{2x + 1}{x - 3} \)
  • \( g(x) = \frac{x^2 - 4}{x^2 + 3x + 2} \)
  • \( h(x) = \frac{5}{x^2 + 1} \)

Non-Examples:

  • \( f(x) = \sqrt{x} \) (not a polynomial in denominator)
  • \( g(x) = \frac{\sin x}{x} \) (numerator not polynomial)

[ \( f(x) = \frac{1}{x} \) ]

Rational functions often have asymptotes and discontinuities

Domain of Rational Functions

The domain of a rational function consists of all real numbers except those that make the denominator zero.

To find the domain:

  1. Set the denominator equal to zero: \( Q(x) = 0 \)
  2. Solve for \( x \)
  3. Exclude these values from the real numbers

Example: Find domain of \( f(x) = \frac{x + 2}{x^2 - 9} \)

Solution:

Set denominator equal to zero:

\[ x^2 - 9 = 0 \] \[ x^2 = 9 \] \[ x = \pm 3 \]

Domain: All real numbers except \( x = 3 \) and \( x = -3 \)

\[ \text{Domain} = \{x \in \mathbb{R} \mid x \neq -3, x \neq 3\} \]

Interval notation: \( (-\infty, -3) \cup (-3, 3) \cup (3, \infty) \)

Vertical Asymptotes:

The values excluded from the domain often correspond to vertical asymptotes in the graph of the function.

Range of Rational Functions

The range of a rational function is the set of all possible output values (y-values). Determining the range is often more complex than finding the domain.

Method to find range:

  1. Express the function as \( y = \frac{P(x)}{Q(x)} \)
  2. Rearrange to form a quadratic in \( x \): \( yQ(x) = P(x) \)
  3. For real \( x \), the discriminant must be ≥ 0
  4. Solve the inequality for \( y \)

Example: Find range of \( f(x) = \frac{1}{x - 2} \)

Solution:

Let \( y = \frac{1}{x - 2} \)

Solve for \( x \):

\[ y(x - 2) = 1 \] \[ x - 2 = \frac{1}{y} \] \[ x = \frac{1}{y} + 2 \]

Since \( x \) must be real and \( x \neq 2 \), \( y \) can be any real number except 0.

\[ \text{Range} = \{y \in \mathbb{R} \mid y \neq 0\} \]

Horizontal Asymptotes:

The behavior of the function as \( x \) approaches ±∞ often determines horizontal asymptotes, which may affect the range.

Zeroes of Rational Functions

The zeroes (or roots) of a rational function are the x-values where the function equals zero. These occur where the numerator is zero (and the denominator is not zero).

To find zeroes:

  1. Set the numerator equal to zero: \( P(x) = 0 \)
  2. Solve for \( x \)
  3. Verify these values don't make the denominator zero

Example: Find zeroes of \( f(x) = \frac{x^2 - 4}{x^2 - x - 6} \)

Solution:

Set numerator equal to zero:

\[ x^2 - 4 = 0 \] \[ x^2 = 4 \] \[ x = \pm 2 \]

Check denominator at these points:

For \( x = 2 \): \( (2)^2 - (2) - 6 = -4 \neq 0 \)

For \( x = -2 \): \( (-2)^2 - (-2) - 6 = 0 \) → Denominator zero!

Therefore, the only zero is at \( x = 2 \)

Holes vs. Zeroes:

If a value makes both numerator and denominator zero, there's a hole (removable discontinuity) rather than a zero.

Operations on Rational Functions

Addition and Subtraction

To add/subtract rational functions:

  1. Find a common denominator (usually the product of denominators)
  2. Rewrite each fraction with the common denominator
  3. Combine numerators
  4. Simplify the result

Example: Add \( \frac{2}{x + 1} + \frac{3}{x - 2} \)

Solution:

Common denominator: \( (x + 1)(x - 2) \)

\[ \frac{2(x - 2) + 3(x + 1)}{(x + 1)(x - 2)} = \frac{2x - 4 + 3x + 3}{(x + 1)(x - 2)} \] \[ = \frac{5x - 1}{(x + 1)(x - 2)} \]

Multiplication

To multiply rational functions:

  1. Multiply numerators together
  2. Multiply denominators together
  3. Simplify by factoring and canceling common factors

Example: Multiply \( \frac{x - 2}{x + 3} \times \frac{x^2 - 9}{x^2 - 4} \)

Solution:

\[ \frac{(x - 2)(x^2 - 9)}{(x + 3)(x^2 - 4)} \]

Factor numerator and denominator:

\[ = \frac{(x - 2)(x - 3)(x + 3)}{(x + 3)(x - 2)(x + 2)} \]

Cancel common factors:

\[ = \frac{x - 3}{x + 2} \quad \text{for} \quad x \neq -3, x \neq 2 \]

Division

To divide rational functions:

  1. Multiply by the reciprocal of the divisor
  2. Follow multiplication steps

Example: Divide \( \frac{x^2 - 4}{x^2 + 3x + 2} \div \frac{x - 2}{x + 1} \)

Solution:

\[ \frac{x^2 - 4}{x^2 + 3x + 2} \times \frac{x + 1}{x - 2} \]

Factor all polynomials:

\[ = \frac{(x - 2)(x + 2)}{(x + 1)(x + 2)} \times \frac{x + 1}{x - 2} \]

Cancel common factors:

\[ = 1 \quad \text{for} \quad x \neq -2, x \neq -1, x \neq 2 \]

Partial Fractions

Partial fraction decomposition is the process of breaking down a complex rational expression into simpler fractions that can be more easily integrated or manipulated.

General Approach:

  1. Factor the denominator completely
  2. Determine the form of the partial fractions based on the factors
  3. Multiply through by the denominator to clear fractions
  4. Solve for the unknown coefficients

Applications:

Partial fractions are essential for:

  • Integration of rational functions
  • Solving differential equations
  • Laplace transforms

Case 1: Distinct Linear Factors

For each distinct linear factor \( (ax + b) \) in the denominator, there's a partial fraction \( \frac{A}{ax + b} \).

Example: Decompose \( \frac{3x + 5}{(x + 1)(x - 2)} \)

Solution:

\[ \frac{3x + 5}{(x + 1)(x - 2)} = \frac{A}{x + 1} + \frac{B}{x - 2} \]

Multiply through by \( (x + 1)(x - 2) \):

\[ 3x + 5 = A(x - 2) + B(x + 1) \]

Solve for A and B by substitution or equating coefficients:

Let \( x = -1 \): \( 3(-1) + 5 = A(-3) \) ⇒ \( A = -\frac{2}{3} \)

Let \( x = 2 \): \( 3(2) + 5 = B(3) \) ⇒ \( B = \frac{11}{3} \)

Thus:

\[ \frac{3x + 5}{(x + 1)(x - 2)} = \frac{-2/3}{x + 1} + \frac{11/3}{x - 2} \]

Factor denominator: \( x^2 - 5x + 6 = (x - 2)(x - 3) \)

Denominator zero when \( x = 2 \) or \( x = 3 \)

Domain: All real numbers except \( x = 2 \) and \( x = 3 \)

\[ (-\infty, 2) \cup (2, 3) \cup (3, \infty) \]

Question 2

Find the zeroes of \( g(x) = \frac{x^3 - 2x^2 - 5x + 6}{x^2 - 4} \)

Factor numerator: \( x^3 - 2x^2 - 5x + 6 = (x - 1)(x + 2)(x - 3) \)

Factor denominator: \( x^2 - 4 = (x - 2)(x + 2) \)

Numerator zero at \( x = 1, -2, 3 \)

Denominator zero at \( x = 2, -2 \)

Thus, zeroes at \( x = 1 \) and \( x = 3 \) (x = -2 makes denominator zero)

Question 3

Express \( \frac{5x - 1}{(x + 2)(x - 3)} \) in partial fractions

Let \( \frac{5x - 1}{(x + 2)(x - 3)} = \frac{A}{x + 2} + \frac{B}{x - 3} \)

Multiply through:

\[ 5x - 1 = A(x - 3) + B(x + 2) \]

Let \( x = -2 \): \( -10 - 1 = A(-5) \) ⇒ \( A = \frac{11}{5} \)

Let \( x = 3 \): \( 15 - 1 = B(5) \) ⇒ \( B = \frac{14}{5} \)

Thus:

\[ \frac{5x - 1}{(x + 2)(x - 3)} = \frac{11/5}{x + 2} + \frac{14/5}{x - 3} \]

Case 2: Repeated Linear Factors

For a repeated linear factor \( (ax + b)^n \), include terms with all powers up to n:

\[ \frac{A_1}{ax + b} + \frac{A_2}{(ax + b)^2} + \cdots + \frac{A_n}{(ax + b)^n} \]

Example: Decompose \( \frac{x^2 + 1}{(x - 1)^3} \)

Solution:

\[ \frac{x^2 + 1}{(x - 1)^3} = \frac{A}{x - 1} + \frac{B}{(x - 1)^2} + \frac{C}{(x - 1)^3} \]

Multiply through and solve similarly to Case 1.

Case 3: Irreducible Quadratic Factors

For irreducible quadratics \( (ax^2 + bx + c) \), use a linear numerator:

\[ \frac{Ax + B}{ax^2 + bx + c} \]

Example: Decompose \( \frac{2x^2 - x + 4}{(x^2 + 1)(x - 1)} \)

Solution:

\[ \frac{2x^2 - x + 4}{(x^2 + 1)(x - 1)} = \frac{Ax + B}{x^2 + 1} + \frac{C}{x - 1} \]

Multiply through and solve for A, B, and C.
