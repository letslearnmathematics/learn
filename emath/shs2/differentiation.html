Introduction to Differentiation

Differentiation is the process of finding the derivative of a function, which represents the rate of change of the function with respect to its variable.

If \( y = f(x) \), the derivative is denoted as:

\[ \frac{dy}{dx} \quad \text{or} \quad f'(x) \]

Geometric Interpretation:

The derivative represents the slope of the tangent line to the curve at any point.

Graph showing tangent line to a curve

Physical Interpretation:

In physics, the derivative of position with respect to time gives velocity.

\[ v = \frac{ds}{dt} \]

Key Concepts

  • Derivative: Instantaneous rate of change
  • Notation: Leibniz (\( \frac{dy}{dx} \)) or Prime (\( f'(x) \))
  • Applications: Tangent lines, optimization, motion problems

Differentiation from First Principle

The first principle (or definition) of differentiation uses limits to find the exact gradient at a point:

\[ f'(x) = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h} \]

Example: Differentiate \( f(x) = x^2 \) from first principles

Step 1: Apply the definition

\[ f'(x) = \lim_{h \to 0} \frac{(x+h)^2 - x^2}{h} \]

Step 2: Expand and simplify

\[ = \lim_{h \to 0} \frac{x^2 + 2xh + h^2 - x^2}{h} \] \[ = \lim_{h \to 0} \frac{2xh + h^2}{h} \] \[ = \lim_{h \to 0} (2x + h) \]

Step 3: Take the limit

\[ f'(x) = 2x \]

Why First Principle?

  • Gives the fundamental definition
  • Used to derive all other rules
  • Important for understanding the concept

Rules of Differentiation

1. Power Rule

For \( f(x) = x^n \):

\[ f'(x) = nx^{n-1} \]

Example:

\[ \text{If } y = x^4, \text{ then } \frac{dy}{dx} = 4x^3 \]

2. Constant Rule

The derivative of a constant is zero:

\[ \frac{d}{dx}(c) = 0 \]

Example:

\[ \frac{d}{dx}(5) = 0 \]

3. Sum Rule

Derivative of a sum is sum of derivatives:

\[ \frac{d}{dx}[f(x) + g(x)] = f'(x) + g'(x) \]

Example:

\[ \text{If } y = x^2 + 3x, \text{ then } \frac{dy}{dx} = 2x + 3 \]

4. Product Rule

For \( y = u \times v \):

\[ \frac{dy}{dx} = u\frac{dv}{dx} + v\frac{du}{dx} \]

Example:

\[ \text{If } y = x^2 \sin x, \text{ then } \] \[ \frac{dy}{dx} = x^2 \cos x + 2x \sin x \]

5. Quotient Rule

For \( y = \frac{u}{v} \):

\[ \frac{dy}{dx} = \frac{v\frac{du}{dx} - u\frac{dv}{dx}}{v^2} \]

Example:

\[ \text{If } y = \frac{x}{x^2 + 1}, \text{ then } \] \[ \frac{dy}{dx} = \frac{(x^2+1)(1) - x(2x)}{(x^2+1)^2} \]

6. Chain Rule

For composite functions \( y = f(g(x)) \):

\[ \frac{dy}{dx} = f'(g(x)) \times g'(x) \]

Example:

\[ \text{If } y = (3x^2 + 2)^4, \text{ then } \] \[ \frac{dy}{dx} = 4(3x^2 + 2)^3 \times 6x \]

Differentiation of Implicit Functions

Implicit differentiation is used when functions are not expressed explicitly as y = f(x).

Example: Find \( \frac{dy}{dx} \) for \( x^2 + y^2 = 25 \)

Step 1: Differentiate both sides with respect to x

\[ \frac{d}{dx}(x^2) + \frac{d}{dx}(y^2) = \frac{d}{dx}(25) \]

Step 2: Apply chain rule to y terms

\[ 2x + 2y\frac{dy}{dx} = 0 \]

Step 3: Solve for \( \frac{dy}{dx} \)

\[ 2y\frac{dy}{dx} = -2x \] \[ \frac{dy}{dx} = -\frac{x}{y} \]

When to Use Implicit Differentiation:

  • When y cannot be easily isolated
  • For equations like circles, ellipses
  • For inverse functions

Differentiation of sin x and cos x

The derivatives of basic trigonometric functions:

Sine Function

\[ \frac{d}{dx}(\sin x) = \cos x \]

Example:

\[ \text{If } y = \sin(3x), \text{ then } \] \[ \frac{dy}{dx} = 3\cos(3x) \text{ (using chain rule)} \]

Cosine Function

\[ \frac{d}{dx}(\cos x) = -\sin x \]

Example:

\[ \text{If } y = x^2 \cos x, \text{ then } \] \[ \frac{dy}{dx} = 2x \cos x - x^2 \sin x \text{ (using product rule)} \]

Practice Exercise

Question 1

Differentiate \( y = 3x^4 - 2x^3 + 5x - 7 \)

\[ \frac{dy}{dx} = 12x^3 - 6x^2 + 5 \]

Question 2

Find the derivative of \( y = (2x + 1)^5 \) using chain rule

\[ \frac{dy}{dx} = 5(2x + 1)^4 \times 2 = 10(2x + 1)^4 \]

Question 3

Use implicit differentiation to find \( \frac{dy}{dx} \) for \( x^3 + y^3 = 6xy \)

Differentiating both sides:

\[ 3x^2 + 3y^2\frac{dy}{dx} = 6y + 6x\frac{dy}{dx} \]

Rearrange:

\[ \frac{dy}{dx} = \frac{6y - 3x^2}{3y^2 - 6x} = \frac{2y - x^2}{y^2 - 2x} \]
