Using the Protractor

You can use the protractor to measure the size of angles. You can also use the protractor to draw angles. In this lesson, we will learn how to use the protractor to draw angles such as 30°, 45°, 60°, 75°, 90°, 120°, 150°, 270° and 300°.

The video below describes how to use the protractor when drawing and measuring angles.
